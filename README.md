# 📚 Stack Docs

Open-source fullstack web development learning platform.

**Live**: https://stack-lovat-eight.vercel.app

## Quick Start

```bash
npm install
npm run dev
```

## Features

- 📖 Interactive MDX documentation
- 🔐 Admin panel with authentication
- 🔍 Full-text search
- 💾 Supabase database integration
- 🌙 Dark mode support
- 📱 Responsive design
