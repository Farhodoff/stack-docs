# Context API va Global State

React Context API - komponentlar daraxtining har qanday joyida ma'lumotlarni ulashing imkonini beruvchi mexanizm. Bu prop drilling muammosini hal qiladi va global state boshqarish uchun ishlatiladi.

## 1. Prop Drilling Muammosi

### Muammo:

```jsx
// Yuqori darajadagi komponent
function App() {
  const [user, setUser] = useState({
    name: 'Ali Valiyev',
    theme: 'dark',
    language: 'uz'
  });

  return (
    <div>
      <Header user={user} />
      <MainContent user={user} />
      <Footer user={user} />
    </div>
  );
}

function Header({ user }) {
  return (
    <header>
      <Navigation user={user} />
    </header>
  );
}

// Navigation komponentiga prop drilling
function Navigation({ user }) {
  return (
    <nav>
      <ProfileMenu user={user} />
      <ThemeToggle theme={user.theme} />
    </nav>
  );
}

// Oxir-oqibat bu yerda foydalaniladi
function ProfileMenu({ user }) {
  return (
    <div>
      <span>Salom, {user.name}!</span>
      <UserDropdown user={user} />
    </div>
  );
}

function UserDropdown({ user }) {
  return (
    <div>
      <p>Ism: {user.name}</p>
      <p>Til: {user.language}</p>
      <p>Mavzu: {user.theme}</p>
    </div>
  );
}
```

**Muammo:** MA'lumot faqat oxirgi komponentda kerak, lekin barcha oraliq komponentlar orqali uzatilishi kerak!

## 2. Context API yechimi

### Asosiy Context yaratish:

```jsx
import React, { createContext, useContext, useState } from 'react';

// 1. Context yaratish
const UserContext = createContext();

// 2. Context Provider komponenti
export function UserProvider({ children }) {
  const [user, setUser] = useState({
    name: 'Ali Valiyev',
    email: 'ali@example.com',
    theme: 'light',
    language: 'uz',
    avatar: 'https://via.placeholder.com/150'
  });

  const updateUser = (updates) => {
    setUser(prev => ({ ...prev, ...updates }));
  };

  const toggleTheme = () => {
    setUser(prev => ({
      ...prev,
      theme: prev.theme === 'light' ? 'dark' : 'light'
    }));
  };

  const changeLanguage = (language) => {
    setUser(prev => ({ ...prev, language }));
  };

  const value = {
    user,
    updateUser,
    toggleTheme,
    changeLanguage
  };

  return (
    <UserContext.Provider value={value}>
      {children}
    </UserContext.Provider>
  );
}

// 3. Custom hook context uchun
export function useUser() {
  const context = useContext(UserContext);

  if (!context) {
    throw new Error('useUser useUserProvider ichida ishlatilishi kerak');
  }

  return context;
}

// 4. Context'ni ishlatish
function App() {
  return (
    <UserProvider>
      <div>
        <Header />       {/* Props yuq! */}
        <MainContent />  {/* Props yuq! */}
        <Footer />       {/* Props yuq! */}
      </div>
    </UserProvider>
  );
}

function Header() {
  return (
    <header>
      <Navigation />   {/* Props yuq! */}
    </header>
  );
}

function Navigation() {
  return (
    <nav>
      <ProfileMenu />  {/* Props yuq! */}
      <ThemeToggle />  {/* Props yuq! */}
    </nav>
  );
}

function ProfileMenu() {
  const { user } = useUser();  // To'g'ridan-to'g'ri context'dan

  return (
    <div>
      <img src={user.avatar} alt={user.name} />
      <span>Salom, {user.name}!</span>
    </div>
  );
}

function ThemeToggle() {
  const { user, toggleTheme } = useUser();

  return (
    <button onClick={toggleTheme}>
      {user.theme === 'light' ? '🌙' : '☀️'}
      Mavzu: {user.theme}
    </button>
  );
}
```

## 3. Multiple Context'lar

Real loyihada bir nechta context bo'lishi mumkin:

```jsx
// Theme Context
const ThemeContext = createContext();

export function ThemeProvider({ children }) {
  const [theme, setTheme] = useState('light');
  const [colors, setColors] = useState({
    primary: '#007bff',
    secondary: '#6c757d',
    success: '#28a745',
    danger: '#dc3545'
  });

  const toggleTheme = () => {
    setTheme(prev => prev === 'light' ? 'dark' : 'light');
  };

  const updateColors = (newColors) => {
    setColors(prev => ({ ...prev, ...newColors }));
  };

  return (
    <ThemeContext.Provider value={{
      theme,
      colors,
      toggleTheme,
      updateColors
    }}>
      {children}
    </ThemeContext.Provider>
  );
}

export function useTheme() {
  const context = useContext(ThemeContext);
  if (!context) {
    throw new Error('useTheme useThemeProvider ichida ishlatilishi kerak');
  }
  return context;
}

// Language Context
const LanguageContext = createContext();

export function LanguageProvider({ children }) {
  const [language, setLanguage] = useState('uz');
  const [translations, setTranslations] = useState({
    uz: {
      welcome: 'Xush kelibsiz',
      login: 'Kirish',
      logout: 'Chiqish',
      settings: 'Sozlamalar'
    },
    en: {
      welcome: 'Welcome',
      login: 'Login',
      logout: 'Logout',
      settings: 'Settings'
    },
    ru: {
      welcome: 'Добро пожаловать',
      login: 'Вход',
      logout: 'Выход',
      settings: 'Настройки'
    }
  });

  const t = (key) => {
    return translations[language]?.[key] || key;
  };

  const changeLanguage = (newLanguage) => {
    setLanguage(newLanguage);
  };

  return (
    <LanguageContext.Provider value={{
      language,
      translations,
      t,
      changeLanguage
    }}>
      {children}
    </LanguageContext.Provider>
  );
}

export function useLanguage() {
  const context = useContext(LanguageContext);
  if (!context) {
    throw new Error('useLanguage useLanguageProvider ichida ishlatilishi kerak');
  }
  return context;
}

// Shopping Cart Context
const CartContext = createContext();

export function CartProvider({ children }) {
  const [cartItems, setCartItems] = useState([]);
  const [isOpen, setIsOpen] = useState(false);

  const addToCart = (product) => {
    setCartItems(prev => {
      const existingItem = prev.find(item => item.id === product.id);

      if (existingItem) {
        return prev.map(item =>
          item.id === product.id
            ? { ...item, quantity: item.quantity + 1 }
            : item
        );
      }

      return [...prev, { ...product, quantity: 1 }];
    });
  };

  const removeFromCart = (productId) => {
    setCartItems(prev => prev.filter(item => item.id !== productId));
  };

  const updateQuantity = (productId, quantity) => {
    if (quantity <= 0) {
      removeFromCart(productId);
      return;
    }

    setCartItems(prev =>
      prev.map(item =>
        item.id === productId
          ? { ...item, quantity }
          : item
      )
    );
  };

  const clearCart = () => {
    setCartItems([]);
  };

  const getTotalPrice = () => {
    return cartItems.reduce((total, item) => total + (item.price * item.quantity), 0);
  };

  const getTotalItems = () => {
    return cartItems.reduce((total, item) => total + item.quantity, 0);
  };

  const toggleCart = () => {
    setIsOpen(!isOpen);
  };

  return (
    <CartContext.Provider value={{
      cartItems,
      isOpen,
      addToCart,
      removeFromCart,
      updateQuantity,
      clearCart,
      getTotalPrice,
      getTotalItems,
      toggleCart
    }}>
      {children}
    </CartContext.Provider>
  );
}

export function useCart() {
  const context = useContext(CartContext);
  if (!context) {
    throw new Error('useCart useCartProvider ichida ishlatilishi kerak');
  }
  return context;
}

// Root App - barcha providerlar
function App() {
  return (
    <ThemeProvider>
      <LanguageProvider>
        <UserProvider>
          <CartProvider>
            <MainApp />
          </CartProvider>
        </UserProvider>
      </LanguageProvider>
    </ThemeProvider>
  );
}

function MainApp() {
  const { theme } = useTheme();
  const { language } = useLanguage();

  return (
    <div
      className={`app ${theme}`}
      style={{
        minHeight: '100vh',
        backgroundColor: theme === 'dark' ? '#121212' : '#ffffff',
        color: theme === 'dark' ? '#ffffff' : '#000000'
      }}
    >
      <Header />
      <main>
        <ProductSection />
        <CartSection />
      </main>
      <Footer />
    </div>
  );
}
```

## 4. Context Consumer va Hook Pattern

### Consumer pattern (eski usul):

```jsx
function OldSchoolConsumer() {
  return (
    <UserContext.Consumer>
      {({ user, updateUser }) => (
        <div>
          <h2>{user.name}</h2>
          <button onClick={() => updateUser({ name: 'Yangi nom' })}>
            Nomni o'zgartirish
          </button>
        </div>
      )}
    </UserContext.Consumer>
  );
}
```

### Hook pattern (zamonaviy usul):

```jsx
function ModernHookUsage() {
  const { user, updateUser } = useUser();

  const updateName = () => {
    const newName = prompt('Yangi nomni kiriting:', user.name);
    if (newName) {
      updateUser({ name: newName });
    }
  };

  return (
    <div>
      <h2>{user.name}</h2>
      <button onClick={updateName}>
        Nomni o'zgartirish
      </button>
    </div>
  );
}
```

## 5. Real-world Example: E-commerce App

To'liq e-commerce context tizimini yaratish:

```jsx
// Products Context
const ProductsContext = createContext();

export function ProductsProvider({ children }) {
  const [products] = useState([
    { id: 1, name: 'iPhone 15', price: 999, category: 'phone', image: 'iphone.jpg' },
    { id: 2, name: 'MacBook Pro', price: 1999, category: 'laptop', image: 'macbook.jpg' },
    { id: 3, name: 'AirPods Pro', price: 249, category: 'audio', image: 'airpods.jpg' }
  ]);

  const [searchTerm, setSearchTerm] = useState('');
  const [selectedCategory, setSelectedCategory] = useState('all');
  const [sortBy, setSortBy] = useState('name');

  const filteredProducts = products
    .filter(product =>
      product.name.toLowerCase().includes(searchTerm.toLowerCase()) &&
      (selectedCategory === 'all' || product.category === selectedCategory)
    )
    .sort((a, b) => {
      if (sortBy === 'price') return a.price - b.price;
      return a.name.localeCompare(b.name);
    });

  return (
    <ProductsContext.Provider value={{
      products,
      filteredProducts,
      searchTerm,
      setSearchTerm,
      selectedCategory,
      setSelectedCategory,
      sortBy,
      setSortBy
    }}>
      {children}
    </ProductsContext.Provider>
  );
}

export function useProducts() {
  const context = useContext(ProductsContext);
  if (!context) {
    throw new Error('useProducts useProductsProvider ichida ishlatilishi kerak');
  }
  return context;
}

// E-commerce komponentlari
function ProductGrid() {
  const { filteredProducts } = useProducts();
  const { addToCart } = useCart();
  const { theme } = useTheme();
  const { t } = useLanguage();

  return (
    <div style={{
      display: 'grid',
      gridTemplateColumns: 'repeat(auto-fill, minmax(300px, 1fr))',
      gap: '20px',
      padding: '20px'
    }}>
      {filteredProducts.map(product => (
        <div
          key={product.id}
          style={{
            border: `1px solid ${theme === 'dark' ? '#444' : '#ddd'}`,
            borderRadius: '8px',
            padding: '15px',
            backgroundColor: theme === 'dark' ? '#2d2d2d' : '#fff'
          }}
        >
          <h3>{product.name}</h3>
          <p style={{ fontSize: '20px', color: '#28a745' }}>
            ${product.price}
          </p>
          <button
            onClick={() => addToCart(product)}
            style={{
              width: '100%',
              padding: '10px',
              backgroundColor: '#007bff',
              color: 'white',
              border: 'none',
              borderRadius: '4px',
              cursor: 'pointer'
            }}
          >
            🛒 {t('add_to_cart') || 'Savotga qo\'shish'}
          </button>
        </div>
      ))}
    </div>
  );
}

function ProductFilters() {
  const {
    searchTerm,
    setSearchTerm,
    selectedCategory,
    setSelectedCategory,
    sortBy,
    setSortBy
  } = useProducts();
  const { t } = useLanguage();

  return (
    <div style={{
      display: 'flex',
      gap: '15px',
      padding: '20px',
      backgroundColor: '#f8f9fa',
      borderRadius: '8px',
      margin: '20px'
    }}>
      <input
        type="text"
        value={searchTerm}
        onChange={(e) => setSearchTerm(e.target.value)}
        placeholder={t('search') || 'Qidirish...'}
        style={{ flex: 1, padding: '10px' }}
      />

      <select
        value={selectedCategory}
        onChange={(e) => setSelectedCategory(e.target.value)}
        style={{ padding: '10px' }}
      >
        <option value="all">{t('all_categories') || 'Barcha kategoriyalar'}</option>
        <option value="phone">{t('phones') || 'Telefonlar'}</option>
        <option value="laptop">{t('laptops') || 'Noutbuklar'}</option>
        <option value="audio">{t('audio') || 'Audio'}</option>
      </select>

      <select
        value={sortBy}
        onChange={(e) => setSortBy(e.target.value)}
        style={{ padding: '10px' }}
      >
        <option value="name">{t('sort_by_name') || 'Nomi bo\'yicha'}</option>
        <option value="price">{t('sort_by_price') || 'Narx bo\'yicha'}</option>
      </select>
    </div>
  );
}

function CartSummary() {
  const { cartItems, getTotalPrice, getTotalItems, toggleCart } = useCart();
  const { theme } = useTheme();
  const { t } = useLanguage();

  return (
    <div
      onClick={toggleCart}
      style={{
        position: 'fixed',
        top: '20px',
        right: '20px',
        backgroundColor: theme === 'dark' ? '#2d2d2d' : '#fff',
        border: `2px solid ${theme === 'dark' ? '#444' : '#ddd'}`,
        borderRadius: '8px',
        padding: '15px',
        cursor: 'pointer',
        boxShadow: '0 2px 8px rgba(0,0,0,0.1)'
      }}
    >
      <div style={{ display: 'flex', alignItems: 'center', gap: '10px' }}>
        <span style={{ fontSize: '24px' }}>🛒</span>
        <div>
          <div style={{ fontWeight: 'bold' }}>
            {getTotalItems()} {t('items') || 'ta mahsulot'}
          </div>
          <div style={{ color: '#28a745' }}>
            ${getTotalPrice().toFixed(2)}
          </div>
        </div>
      </div>
    </div>
  );
}

function CartDropdown() {
  const {
    cartItems,
    isOpen,
    removeFromCart,
    updateQuantity,
    clearCart,
    getTotalPrice
  } = useCart();
  const { theme } = useTheme();
  const { t } = useLanguage();

  if (!isOpen) return null;

  return (
    <div style={{
      position: 'fixed',
      top: '120px',
      right: '20px',
      width: '400px',
      maxHeight: '500px',
      backgroundColor: theme === 'dark' ? '#2d2d2d' : '#fff',
      border: `1px solid ${theme === 'dark' ? '#444' : '#ddd'}`,
      borderRadius: '8px',
      boxShadow: '0 4px 12px rgba(0,0,0,0.15)',
      overflowY: 'auto',
      zIndex: 1000

      <div style={{ padding: '15px', borderBottom: '1px solid #eee' }}>
        <h3>{t('shopping_cart') || 'Savat'}</h3>
      </div>

      {cartItems.length === 0 ? (
        <div style={{ padding: '40px', textAlign: 'center', color: '#666' }}>
          {t('cart_empty') || 'Savat bo\'sh'}
        </div>
      ) : (
        <div>
          {cartItems.map(item => (
            <div
              key={item.id}
              style={{
                display: 'flex',
                alignItems: 'center',
                padding: '15px',
                borderBottom: '1px solid #eee'
              }}
            >
              <div style={{ flex: 1 }}>
                <h4 style={{ margin: '0 0 5px 0' }}>{item.name}</h4>
                <p style={{ margin: '0', color: '#666' }}>
                  ${item.price} × {item.quantity}
                </p>
              </div>

              <div style={{ display: 'flex', alignItems: 'center', gap: '10px' }}>
                <button
                  onClick={() => updateQuantity(item.id, item.quantity - 1)}
                  style={{ width: '30px', height: '30px' }}
                >
                  -
                </button>
                <span>{item.quantity}</span>
                <button
                  onClick={() => updateQuantity(item.id, item.quantity + 1)}
                  style={{ width: '30px', height: '30px' }}
                >
                  +
                </button>
                <button
                  onClick={() => removeFromCart(item.id)}
                  style={{ color: '#dc3545', marginLeft: '10px' }}
                >
                  🗑️
                </button>
              </div>
            </div>
          ))}

          <div style={{ padding: '15px' }}>
            <div style={{
              display: 'flex',
              justifyContent: 'space-between',
              fontSize: '18px',
              fontWeight: 'bold',
              marginBottom: '15px'
            }}>
              <span>{t('total') || 'Jami'}:</span>
              <span>${getTotalPrice().toFixed(2)}</span>
            </div>

            <div style={{ display: 'flex', gap: '10px' }}>
              <button
                onClick={clearCart}
                style={{
                  flex: 1,
                  padding: '10px',
                  backgroundColor: '#dc3545',
                  color: 'white',
                  border: 'none',
                  borderRadius: '4px'
                }}
              >
                {t('clear_cart') || 'Tozalash'}
              </button>
              <button
                style={{
                  flex: 2,
                  padding: '10px',
                  backgroundColor: '#28a745',
                  color: 'white',
                  border: 'none',
                  borderRadius: '4px'
                }}
              >
                {t('checkout') || 'To\'lov qilish'}
              </button>
            </div>
          </div>
        </div>
      )}
    </div>
  );
}

// Root component
function EcommerceApp() {
```

## Xulosa va Best Practices

### ✅ Context API best practices:
1. **Small contexts** - kichik va focused context'lar yaratish
2. **Custom hooks** - context uchun custom hook yaratish
3. **Error boundaries** - context mavjud emasligini tekshirish
4. **Provider composition** - context'larni to'g'ri wrapped qilish
5. **Performance** - re-rendering'ni kamaytirish

### 🎯 Qachon Context ishlatish:
1. **Prop drilling** - ko'p darajali props uzatish
2. **Global state** - theme, language, auth
3. **Configuration** - app config, features flags
4. **UI state** - modal, notification states

### ❌ Context'ni ortiqcha ishlatmaslik:
1. **Local state** uchun emas
2. **Server state** uchun emas (React Query/SWR ishlatish)
3. **Performance critical** ma'lumotlar uchun ehtiyot
4. **Frequently changing** data uchun

### 📚 Keyingi mavzular:
1. **Custom hooks** - logic'ni qayta ishlatish
2. **State management libraries** - Redux, Zustand
3. **Performance optimization** - useMemo, useCallback
4. **Testing** - context'li komponentlarni test qilish

Context API React'da global state management uchun eng asosiy va kuchli vosila hisoblanadi!
