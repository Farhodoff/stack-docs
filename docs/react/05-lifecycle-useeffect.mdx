# useEffect va Lifecycle

`useEffect` - React'ning eng muhim hooklaridan biri bo'lib, komponentlarda side effect'larni boshqarish uchun ishlatiladi. U API so'rovlari, DOM manipulatsiya, timer'lar va boshqa tashqi ta'sirlar uchun javobgardir.

## 1. useEffect nima?

useEffect side effect'lar uchun ishlatiladi:
- **API so'rovlari** - ma'lumotlarni olish
- **Event listener'lar** - DOM eventlarini tinglash
- **Timer'lar** - setInterval, setTimeout
- **DOM manipulatsiya** - to'g'ridan-to'g'ri DOM o'zgartirish
- **Cleanup** - resurslarni tozalash

### Class component lifecycle analogi:

| useEffect | Class Methods |
|-----------|---------------|
| `useEffect(() => {})` | `componentDidUpdate` |
| `useEffect(() => {}, [])` | `componentDidMount` |
| `useEffect(() => () => {})` | `componentWillUnmount` |

## 2. useEffect asoslari

### Har render'da ishlash:

```jsx
import React, { useState, useEffect } from 'react';

function EveryRenderExample() {
  const [count, setCount] = useState(0);
  const [name, setName] = useState('');

  // Har render'da ishlaydi (ehtiyot bo'ling!)
  useEffect(() => {
    console.log('Komponent render bo\'ldi!');
    document.title = `Count: ${count}`;
  });

  return (
    <div>
      <h2>Har render'da useEffect</h2>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Count oshir</button>

      <input
        value={name}
        onChange={(e) => setName(e.target.value)}
        placeholder="Ismingiz"
      />

      <p>Console'ni tekshiring - har o'zgarishda log chiqadi</p>
    </div>
  );
}
```

```jsx
function MountOnlyExample() {
```

## 4. Cleanup Functions

Resurslarni tozalash uchun cleanup function ishlatiladi.

### Timer cleanup:

```jsx
function TimerExample() {
  const [seconds, setSeconds] = useState(0);
  const [isRunning, setIsRunning] = useState(false);

  useEffect(() => {
    let intervalId;

    if (isRunning) {
      intervalId = setInterval(() => {
        setSeconds(prevSeconds => prevSeconds + 1);
      }, 1000);
    }

    // Cleanup function
    return () => {
      if (intervalId) {
        clearInterval(intervalId);
        console.log('Interval tozalandi');
      }
    };
  }, [isRunning]); // isRunning dependency

  const handleStart = () => setIsRunning(true);
  const handleStop = () => setIsRunning(false);
  const handleReset = () => {
    setIsRunning(false);
    setSeconds(0);
  };

  const formatTime = (totalSeconds) => {
    const hours = Math.floor(totalSeconds / 3600);
    const minutes = Math.floor((totalSeconds % 3600) / 60);
    const secs = totalSeconds % 60;

    return `${hours.toString().padStart(2, '0')}:${minutes.toString().padStart(2, '0')}:${secs.toString().padStart(2, '0')}`;
  };

  return (
    <div>
      <h2>Timer Example</h2>
      <div style={{ fontSize: '2em', fontFamily: 'monospace' }}>
        {formatTime(seconds)}
      </div>

      <div>
        <button onClick={handleStart} disabled={isRunning}>
          Boshlash
        </button>
        <button onClick={handleStop} disabled={!isRunning}>
          To'xtatish
        </button>
        <button onClick={handleReset}>Reset</button>
      </div>

      <p>Status: {isRunning ? 'Ishlayapti' : "To'xtatilgan"}</p>
    </div>
  );
}
```

### Event listener cleanup:

```jsx
function WindowSizeTracker() {
  const [windowSize, setWindowSize] = useState({
    width: typeof window !== 'undefined' ? window.innerWidth : 0,
    height: typeof window !== 'undefined' ? window.innerHeight : 0
  });

  useEffect(() => {
    function handleResize() {
      setWindowSize({
        width: window.innerWidth,
        height: window.innerHeight
      });
    }

    // Event listener qo'shish
    window.addEventListener('resize', handleResize);

    // Cleanup function - event listener'ni o'chirish
    return () => {
      window.removeEventListener('resize', handleResize);
      console.log('Window resize listener o\'chirildi');
    };
  }, []); // Bo'sh dependency - faqat mount/unmount da

  return (
    <div>
      <h2>Window Size Tracker</h2>

      <p>Height: {windowSize.height}px</p>
      <p>Oynani kengligini o'zgartiring va natijani kuzating</p>

  );
}
```

## 5. Data Fetching Patterns

Real API'lar bilan ishlash uchun pattern'lar.

### Async/await pattern:

```jsx
function PostsList() {
  const [posts, setPosts] = useState([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);
  const [page, setPage] = useState(1);

  useEffect(() => {
    let abortController = new AbortController();

    const fetchPosts = async () => {
      try {
        setLoading(true);
        setError(null);

        const response = await fetch(
          `https://jsonplaceholder.typicode.com/posts?_page=${page}&_limit=5`,
          { signal: abortController.signal }
        );

        if (!response.ok) {
          throw new Error(`HTTP error! status: ${response.status}`);
        }

        const postsData = await response.json();
        setPosts(postsData);
      } catch (error) {
        if (error.name !== 'AbortError') {
          setError(error.message);
          console.error('Posts yuklash xatoligi:', error);
        }
      } finally {
        if (!abortController.signal.aborted) {
          setLoading(false);
        }
      }
    };

    fetchPosts();

    // Cleanup - request'ni bekor qilish
    return () => {
      abortController.abort();
    };
  }, [page]); // page o'zgarganda yangi so'rov

  const nextPage = () => setPage(page + 1);
  const prevPage = () => setPage(Math.max(1, page - 1));

  if (loading) return <div>Postlar yuklanmoqda...</div>;
  if (error) return <div>Xatolik: {error}</div>;

  return (
    <div>
      <h2>Posts (Sahifa {page})</h2>

      <div>
        {posts.map(post => (
          <div key={post.id} style={{ border: '1px solid #ccc', padding: '10px', margin: '10px 0' }}>
            <h3>{post.title}</h3>
            <p>{post.body}</p>
            <small>Post ID: {post.id}</small>
          </div>
        ))}
      </div>

      <div>
        <button onClick={prevPage} disabled={page === 1}>
          Oldingi sahifa
        </button>
        <span style={{ margin: '0 10px' }}>Sahifa {page}</span>
        <button onClick={nextPage}>Keyingi sahifa</button>
      </div>
    </div>
  );
}
```

### Custom hook for data fetching:

```jsx
// Custom hook
function useApi(url) {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    let abortController = new AbortController();

    const fetchData = async () => {
      try {
        setLoading(true);
        setError(null);

        const response = await fetch(url, {
          signal: abortController.signal
        });

        if (!response.ok) {
          throw new Error(`HTTP error! status: ${response.status}`);
        }

        const result = await response.json();
        setData(result);
      } catch (error) {
        if (error.name !== 'AbortError') {
          setError(error.message);
        }
      } finally {
        if (!abortController.signal.aborted) {
          setLoading(false);
        }
      }
    };

    fetchData();

    return () => {
      abortController.abort();
    };
  }, [url]);

  return { data, loading, error };
}
```

## 7. Performance Optimization

useEffect'ni optimize qilish usullari.

### Debouncing search:

```jsx
function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const handler = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => {
      clearTimeout(handler);
    };
  }, [value, delay]);

  return debouncedValue;
}

function SearchWithDebounce() {
  const [searchTerm, setSearchTerm] = useState('');
  const [results, setResults] = useState([]);
  const [isSearching, setIsSearching] = useState(false);

  const debouncedSearchTerm = useDebounce(searchTerm, 500);

  useEffect(() => {
    if (debouncedSearchTerm) {
      setIsSearching(true);

      const search = async () => {
        try {
          // API search simulation
          await new Promise(resolve => setTimeout(resolve, 1000));

          const mockResults = [
            `${debouncedSearchTerm} - natija 1`,
            `${debouncedSearchTerm} - natija 2`,
            `${debouncedSearchTerm} - natija 3`
          ];

          setResults(mockResults);
        } catch (error) {
          console.error('Qidiruv xatoligi:', error);
        } finally {
          setIsSearching(false);
        }
      };

      search();
    } else {
      setResults([]);
      setIsSearching(false);
    }
  }, [debouncedSearchTerm]);

  return (
    <div>
      <h2>Debounced Search</h2>

      <input
        type="text"
        value={searchTerm}
        onChange={(e) => setSearchTerm(e.target.value)}
        placeholder="Qidirish (500ms kechikish bilan)..."
      />

      <p>Qidirilayotgan so'z: "{debouncedSearchTerm}"</p>

      {isSearching && <p>Qidirilmoqda...</p>}

      <ul>
        {results.map((result, index) => (
          <li key={index}>{result}</li>
        ))}
      </ul>
    </div>
  );
}
```

## Xulosa va Best Practices

### ✅ useEffect best practices:
1. **Cleanup functions** - har doim resurslarni tozalang
2. **Dependency array** - to'g'ri dependency'lar ko'rsating
3. **AbortController** - API so'rovlarini bekor qilish
4. **Debouncing** - ko'p marta chaqiriluvchi operatsiyalar uchun
5. **Custom hooks** - logic'ni qayta ishlatish uchun

### 🎯 Umumiy xatoliklar:
1. **Missing dependencies** - eslint-plugin-react-hooks yordamida aniqlash
2. **Infinite loops** - noto'g'ri dependency array
3. **Memory leaks** - cleanup function'siz timer'lar
4. **Race conditions** - AbortController ishlatmaslik

### 📚 Keyingi mavzular:
1. **Context API** - global state boshqarish
2. **Custom hooks** - logic'ni qayta ishlatish
3. **Performance hooks** - useMemo, useCallback
4. **React Router** - SPA navigation

useEffect React'da side effect'lar boshqarishning asosi bo'lib, real-world ilovalar yaratishda muhim rol o'ynaydi!
