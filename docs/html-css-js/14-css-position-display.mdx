# CSS Position va Display

Position va Display - elementlarni veb-sahifada qanday ko'rinishi va joylashgani haqida asosiy xususiyatlar.

---

## Display Xususiyati

`display` - elementning qanday ko'rinishini boshqaradi.

### 1️⃣ Display: Block

Butun qatorni egallaydi, yangi qatordan boshlaydi:

```css
div {
  display: block;
}

p {
  display: block;
}

h1, h2, h3 {
  display: block;
}
```

**Block elementlar:** `<div>`, `<p>`, `<h1>-<h6>`, `<section>`, `<header>`, `<footer>`

---

### 2️⃣ Display: Inline

Faqat kerakli o'lchamni egallaydi, qator bilan:

```css
span {
  display: inline;
}

a {
  display: inline;
}

strong {
  display: inline;
}
```

**Inline elementlar:** `<span>`, `<a>`, `<strong>`, `<em>`, `<img>`

---

### 3️⃣ Display: Inline-block

Inline kabi, lekin width va height'ni boshqarish mumkin:

```css
button {
  display: inline-block;
  width: 100px;
  height: 40px;
}

.badge {
  display: inline-block;
  padding: 5px 10px;
  background-color: #2563eb;
  color: white;
}
```

---

### 4️⃣ Display: None

Element ko'rinmaydi:

```css
.hidden {
  display: none;
}

.modal.hidden {
  display: none;
}
```

---

### 5️⃣ Display: Flex

Flexbox layout:

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

---

### 6️⃣ Display: Grid

Grid layout:

```css
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

---

## Position Xususiyati

`position` - element qanday joylashgani haqida bilgi beradi.

### 1️⃣ Position: Static (Standart)

Default qiymat - normal alip:

```css
div {
  position: static;
}
```

---

### 2️⃣ Position: Relative

Elementni o'zining joyidan siljitish:

```css
div {
  position: relative;
  top: 20px;       /* Yuqoridan 20px */
  left: 10px;      /* Chapdan 10px */
}
```

**Xususiyatlar:**
- `top` - Yuqoridan
- `bottom` - Pastdan
- `left` - Chapdan
- `right` - O'ngdan

---

### 3️⃣ Position: Absolute

Eng yaqin positioned ota-elementga nisbatan:

```css
.container {
  position: relative;
}

.child {
  position: absolute;
  top: 20px;
  right: 10px;
}
```

---

### 4️⃣ Position: Fixed

Ekranda qat'iy joyda qoladi, scrollda harakat qilmaydi:

```css
.header {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  height: 60px;
  background-color: #2563eb;
  z-index: 1000;
}

body {
  padding-top: 60px;  /* Header uchun bo'sh joy */
}
```

---

### 5️⃣ Position: Sticky

Scroll bo'lishiga qadar normal, keyin fixed bo'ladi:

```css
.nav {
  position: sticky;
  top: 0;
  background-color: white;
  z-index: 100;
}
```

---

## Z-Index (Qavatlar)

Elementlarning depth (qavat) tartibini boshqarish:

```css
.background {
  position: relative;
  z-index: 1;
}

.modal {
  position: fixed;
  z-index: 1000;     /* Modal yuqorida */
}

.dropdown {
  position: absolute;
  z-index: 999;
}
```

**Qoidalar:**
- Katta z-index = yuqorida
- Faqat `position` qiymat (absolute, relative, fixed, sticky) bilan ishlagani

## Display vs Position Jadvali

| Xususiyat | Qiymat | Maqsadi |
|-----------|--------|---------|
| `display` | `block` | Butun qatorni egallash |
| `display` | `inline` | Faqat zarur joyni egallash |
| `display` | `inline-block` | Inline + width/height |
| `display` | `flex` | Flexbox |
| `display` | `grid` | Grid |
| `position` | `relative` | O'zining joyidan siljitish |
| `position` | `absolute` | Ota-elementga nisbatan |
| `position` | `fixed` | Ekranga nisbatan |
| `position` | `sticky` | Scroll'dagacha fixed |

---

## 🎯 Position va Display Qisqacha

- **Display:** Elementning ko'rinishi
- **Position:** Elementning joylashgani
- **Fixed header:** `position: fixed; top: 0; z-index: 1000;`
- **Sticky nav:** `position: sticky; top: 60px;`
- **Modal:** `position: fixed; z-index: 2000;`
- **Z-index:** Qavat tartibini boshqarish
