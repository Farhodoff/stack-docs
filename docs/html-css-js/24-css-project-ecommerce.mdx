# CSS Project - E-commerce Landing Page

Bu darsda biz barcha o'rgangan CSS kontsepyalarini (Flexbox, Grid, animations, responsive, BEM) qo'llab, real-world e-commerce landing page yaratamiz. Bu loyiha production-ready va professional darajada.

---

## 1️⃣ Loyiha Tuzilishi

E-commerce landing page'da quyidagilar bo'ladi:

```
1. Header (Logo + Navigation + Search)
2. Hero Section (Asosiy tasvir va CTA)
3. Featured Products (Xususiy mahsulotlar)
4. Categories (Kategoriyalar)
5. Product Grid (Mahsulotlar to'r)
6. Newsletter Signup (Yangiliklar uchun ro'yxat)
7. Footer (Pastki qism)
```

---

## 2️⃣ Loyyха o'rganish strategiyasi

```
1. HTML struktura (BEM qoidasi bilan)
2. CSS base va reset
3. Responsive layout (Mobile-first)
4. Ranglar va tipografiya
5. Animatsiyalar va effektlar
6. Interaktivlik
```
