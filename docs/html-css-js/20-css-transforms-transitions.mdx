# CSS - Transform va Transition

Transform - elementni o'zgartirishni (aylanish, masshtablash, harakatlash) ta'minlaydi. Transition - o'zgarishlar ornini yumshoq qiladi.

---

## 1️⃣ Transform Nima?

Transform elementni o'zgartiradi, lekin document oqimiga ta'sir qilmaydi:

```css
.box {
  /* Document oqimini o'zgaritmaydi */
  transform: translateX(50px);
}

/* Boshqacha */
.box {
  /* Document oqimini siqadi */
  margin-left: 50px;
}
```

---

## 2️⃣ Translate (Ko'chirish)

Elementni ko'chirib o'tkazish:

```css
/* X o'qi bo'ylab */
.box {
  transform: translateX(50px);   /* 50px o'ngga */
  transform: translateX(-50px);  /* 50px chapga */
}

/* Y o'qi bo'ylab */
.box {
  transform: translateY(50px);   /* 50px pastga */
  transform: translateY(-50px);  /* 50px yuqoriga */
}

/* Ikkalasini birga */
.box {
  transform: translate(50px, 100px);  /* X, Y */
}

/* 3D translate */
.box {
  transform: translateZ(50px);
  transform: translate3d(50px, 100px, 50px);
}
```

---

## 3️⃣ Scale (Masshtablash)

Elementni kattalashtirish yoki kichiklashtirish:

```css
/* X o'qi bo'ylab */
.box {
  transform: scaleX(1.5);   /* 1.5 marta katta */
  transform: scaleX(0.5);   /* Yarim o'lcham */
}

/* Y o'qi bo'ylab */
.box {
  transform: scaleY(1.5);
}

/* Ikkalasini */
.box {
  transform: scale(1.5);      /* Ikkalasini 1.5 marta */
  transform: scale(1.5, 0.8); /* X - 1.5, Y - 0.8 */
}

/* 3D scale */
.box {
  transform: scaleZ(1.5);
  transform: scale3d(1.5, 1.5, 1);
}
```

---

## 4️⃣ Rotate (Aylanish)

Elementni aylantiradish:

```css
/* Tekis aylanish */
.box {
  transform: rotate(45deg);      /* 45 gradus */
  transform: rotate(0.5turn);    /* 0.5 aylanish */
  transform: rotate(3.14rad);    /* Radian */
}

/* 3D aylanish */
.box {
  transform: rotateX(45deg);  /* X o'qi atrofida */
  transform: rotateY(45deg);  /* Y o'qi atrofida */
  transform: rotateZ(45deg);  /* Z o'qi atrofida */
}

/* Ixtiyoriy o'qi atrofida */
.box {
  transform: rotate3d(1, 1, 0, 45deg);  /* X, Y, Z, burchak */
}
```

---

## 5️⃣ Skew (Egilish)

Elementni egilib qo'yish:

```css
/* X o'qi bo'ylab */
.box {
  transform: skewX(45deg);
}

/* Y o'yi bo'ylab */
.box {
  transform: skewY(45deg);
}

/* Ikkalasini */
.box {
  transform: skew(45deg, 20deg);  /* X, Y */
}
```

---

## 6️⃣ Transform Origin (Aylanish Markazi)

Transform qayerdan boshlashini belgilash:

```css
.box {
  transform: rotate(45deg);
  /* Default - markaz */
  transform-origin: center;
}

/* Chap ustun burchakdan */
.box {
  transform-origin: top left;
  transform: rotate(45deg);
}

/* Piksellar bilan */
.box {
  transform-origin: 50px 100px;
  transform: rotate(45deg);
}

/* Protsent bilan */
.box {
  transform-origin: 100% 100%;  /* O'ng pastki burchak */
  transform: rotate(45deg);
}
```

---

## 7️⃣ Transition (O'zgarish Harakati)

O'zgarishlarni yumshoq qilish:

```css
.box {
  background-color: red;

  /* Barcha xususiyatlarni 0.5 sekundda */
  transition: all 0.5s ease;
}

.box:hover {
  background-color: blue;
}

/* Alohida xususiyatlar */
.box {
  transition: background-color 0.3s ease;
  transition: transform 0.5s ease;
}

/* Bir nechta xususiyat */
.box {
  transition:
    background-color 0.3s ease,
    transform 0.5s ease,
    border-radius 0.7s ease;
}
```

---

## 8️⃣ Transition Timing Functions

O'zgarishning tezligi:

```css
.box {
  transition: all 1s linear;      /* Bir xil tezlik */
}

.box {
  transition: all 1s ease;        /* Sekin boshlash va tugash */
}

.box {
  transition: all 1s ease-in;     /* Sekin boshlash */
}

.box {
  transition: all 1s ease-out;    /* Sekin tugash */
}

.box {
  transition: all 1s ease-in-out; /* Ikkalasi */
}

/* Custom bezier */
.box {
  transition: all 1s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

/* Steps */
.box {
  transition: all 1s steps(4);  /* 4 qadam */
}
```

---

## 9️⃣ Transition Delay (O'zgarish Kechikishi)

O'zgarish qachon boshlashini belgilash:

```css
.box {
  transition: all 0.5s ease;

  /* 0.2 sekunddan keyin boshlash */
  transition-delay: 0.2s;
}

/* Alohida */
.box {
  transition-property: background-color;
  transition-duration: 0.5s;
  transition-timing-function: ease;
  transition-delay: 0.2s;
}
```

## 1️⃣1️⃣ Transform Shorthand va Kombinatsiyalar

```css
/* Bir nechta transform */
.box {
  transform: translateX(50px) rotateZ(45deg) scale(1.5);
}

/* Tartibi muhim */
.box {
  /* Ikkala ham boshqacha ko'rinadi */
  transform: translate(50px, 50px) rotate(45deg);
  transform: rotate(45deg) translate(50px, 50px);
}

/* Perspective - 3D */
.parent {
  perspective: 1000px;
}

.child {
  transform: rotateY(45deg);
  transform-style: preserve-3d;
}
```

## 1️⃣2️⃣ Transition vs Animation

| Xususiyat | Transition | Animation |
|-----------|-----------|-----------|
| Boshlash | Hover yoki JS bilan | Avtomatik yoki JS |
| Takrorlanish | Faqat bir marta | Kaç kez tahsin qilish |
| Boshqarish | Oson | Murakkab |
| Timing | simple | Murakkab |

---

## Transform va Transition Cheat Sheet

| Xususiyat | Misol | Maqsadi |
|-----------|-------|---------|
| `translate()` | `translate(50px, 100px)` | Ko'chirish |
| `scale()` | `scale(1.5)` | Masshtablash |
| `rotate()` | `rotate(45deg)` | Aylanish |
| `skew()` | `skewX(30deg)` | Egilish |
| `transform-origin` | `center` | Aylanish markazi |
| `transition` | `all 0.5s ease` | O'zgarish harakati |
| `transition-delay` | `0.2s` | O'zgarish kechikishi |

---

## 🎯 Transform va Transition Qisqacha

- `transform` → Elementni o'zgartirish (translate, scale, rotate, skew)
- `transition` → O'zgarishlarni yumshoq qilish
- `transform-origin` → Transform merkazi
- **TAVSIYA:** UI interaktivlikni oshirish uchun har doim transition ishlating!
