# CSS - Animatsiya (@keyframes)

Animatsiya - elementni bir holatdan boshqa holatga yumshoq o'tkazish. @keyframes qoidasi orqali animatsiyani aniq boshqarish mumkin.

---

## 1️⃣ @keyframes Asoslari

@keyframes orqali animatsiyaning har bir bosqichini belgilash:

```css
/* Animatsiyani belgilash */
@keyframes slidein {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

/* Animatsiyani elementga qo'llash */
.box {
  animation-name: slidein;
  animation-duration: 2s;
}

/* Protsent bilan */
@keyframes slidein {
  0% {
    transform: translateX(-100%);
  }
  50% {
    transform: translateX(-50%);
  }
  100% {
    transform: translateX(0);
  }
}
```

---

## 2️⃣ Animation Duration (Vaqt Davomiyligi)

Animatsiya qancha vaqt ichida bo'lishini belgilash:

```css
.box {
  animation-name: slidein;

  /* 2 sekund */
  animation-duration: 2s;

  /* Millisekundlar */
  animation-duration: 2000ms;
}
```

---

## 3️⃣ Animation Timing Function (Tezlik Funktsiyasi)

Animatsiya tezligi:

```css
.box {
  animation-name: slidein;
  animation-duration: 2s;

  /* Bir xil tezlik */
  animation-timing-function: linear;
}

.box2 {
  /* Sekin boshlash va tugash */
  animation-timing-function: ease;
}

.box3 {
  /* Sekin boshlash */
  animation-timing-function: ease-in;
}

.box4 {
  /* Sekin tugash */
  animation-timing-function: ease-out;
}

.box5 {
  /* Ikkalasi */
  animation-timing-function: ease-in-out;
}

/* Custom */
.box6 {
  animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

/* Steps */
.box7 {
  animation-timing-function: steps(4);
}
```

---

## 4️⃣ Animation Delay (Kechikish)

Animatsiya qachon boshlashini belgilash:

```css
.box {
  animation-name: slidein;
  animation-duration: 2s;

  /* 0.5 sekunddan keyin boshlash */
  animation-delay: 0.5s;
}

/* Salbiy kechikish - animatsiya o'rtasidan boshlash */
.box {
  animation-delay: -1s;  /* 1 sekundni o'tgandek */
}
```

---

## 5️⃣ Animation Iteration Count (Takrorlanish Soni)

Animatsiya qancha marta takrorlanishini belgilash:

```css
.box {
  animation-name: slidein;
  animation-duration: 2s;

  /* Faqat bir marta */
  animation-iteration-count: 1;
}

.box2 {
  /* 3 marta */
  animation-iteration-count: 3;
}

.box3 {
  /* Cheksiz takrorlanish */
  animation-iteration-count: infinite;
}
```

---

## 6️⃣ Animation Direction (Yo'nalishi)

Animatsiya yo'nalishi:

```css
.box {
  animation-name: slidein;
  animation-duration: 2s;
  animation-iteration-count: 2;

  /* Normal - 0% dan 100% ga */
  animation-direction: normal;
}

.box2 {
  /* Teskari - 100% dan 0% ga */
  animation-direction: reverse;
}

.box3 {
  /* Normal, keyin teskari */
  animation-direction: alternate;
}

.box4 {
  /* Teskari, keyin normal */
  animation-direction: alternate-reverse;
}
```

---

## 7️⃣ Animation Fill Mode (To'ldirish Rejimi)

Animatsiya tugaganida element qanday bo'lishi:

```css
.box {
  animation-name: slidein;
  animation-duration: 2s;

  /* Default - animatsiya boshiga qaytadi */
  animation-fill-mode: none;
}

.box2 {
  /* Tugaganida animatsiya oxirgi holatida qoladi */
  animation-fill-mode: forwards;
}

.box3 {
  /* Boslanmasiga animatsiya boshlang'ich holatida qoladi */
  animation-fill-mode: backwards;
}

.box4 {
  /* Ikkalasi */
  animation-fill-mode: both;
}
```

---

## 8️⃣ Animation Shorthand (Qisqacha)

Barcha animatsiya xususiyatlari bir qatorda:

```css
.box {
  /* name duration timing-function delay iteration direction fill-mode */
  animation: slidein 2s ease-in 0.5s 3 normal forwards;
}

/* Qisqacha ko'rinishi */
.box {
  animation-name: slidein;
  animation-duration: 2s;
  animation-timing-function: ease-in;
  animation-delay: 0.5s;
  animation-iteration-count: 3;
  animation-direction: normal;
  animation-fill-mode: forwards;
}
```

## 9️⃣ Praktik Animatsiya Misollari

```css
/* Pulsating effect */
@keyframes pulse {
  0% {
    opacity: 1;
  }
  50% {
    opacity: 0.5;
  }
  100% {
    opacity: 1;
  }
}

/* Bounce effect */
@keyframes bounce {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-30px);
  }
}

/* Rotate */
@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

/* Gradient animation */
@keyframes gradientShift {
  0% {
    background-position: 0% 50%;
  }
  50% {
    background-position: 100% 50%;
  }
  100% {
    background-position: 0% 50%;
  }
}

/* Slide in */
@keyframes slideInLeft {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

/* Fade in and scale */
@keyframes fadeInScale {
  0% {
    opacity: 0;
    transform: scale(0.8);
  }
  100% {
    opacity: 1;
    transform: scale(1);
  }
}
```

## 1️⃣1️⃣ Animation Performance Optimization

Animatsiyani tez va samarali qilish:

```css
/* ✅ YAXSHI - GPU accelerated */
.box {
  animation: slide 1s ease-out;
}

@keyframes slide {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

/* ❌ YOMON - CPU ga bekor */
.box {
  animation: bad 1s ease-out;
}

@keyframes bad {
  from {
    left: -100%;
  }
  to {
    left: 0;
  }
}

/* ✅ YAXSHI - opacity */
.box {
  animation: fade 1s ease-out;
}

@keyframes fade {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

/* ❌ YOMON - width o'zgarishi */
.box {
  animation: grow 1s ease-out;
}

@keyframes grow {
  from {
    width: 0;
  }
  to {
    width: 100%;
  }
}
```

## 1️⃣2️⃣ Animatsiya Cheat Sheet

| Xususiyat | Misol | Maqsadi |
|-----------|-------|---------|
| `@keyframes` | `@keyframes slidein { ... }` | Animatsiyani belgilash |
| `animation-duration` | `2s`, `2000ms` | Vaqt davomiyligi |
| `animation-delay` | `0.5s`, `-1s` | Kechikish |
| `animation-timing-function` | `ease`, `linear`, `ease-in` | Tezlik funktsiyasi |
| `animation-iteration-count` | `3`, `infinite` | Takrorlanish soni |
| `animation-direction` | `normal`, `reverse`, `alternate` | Yo'nalishi |
| `animation-fill-mode` | `forwards`, `backwards`, `both` | Tugaganida holatı |

---

## 🎯 Animatsiya Qisqacha

- `@keyframes` → Animatsiyani belgilash (0%, 50%, 100% yoki from/to)
- `animation` → Animatsiyani qo'llash
- `animation-timing-function` → Tezlik (ease, linear, cubic-bezier)
- **TAVSIYA:** GPU accelerated xususiyatlari (transform, opacity) ishlatilsin!
