# CSS Matn va Fontlar

Matn - web saytning eng muhim elementi. CSS'da tekstni qanday ko'rinishini kontrol qilishimiz mumkin.

---

## 1️⃣ Font Turlari

### System Fontlari

Har bir kompyuterda mavjud bo'lgan fontlar:

```css
body {
  /* Sans-serif - burchaklarsiz */
  font-family: Arial, Helvetica, sans-serif;
}

h1 {
  /* Serif - kichik chiziqlar bor */
  font-family: Georgia, "Times New Roman", serif;
}

code {
  /* Monospace - har bir belgisi bir xil kenglikda */
  font-family: "Courier New", Courier, monospace;
}
```

**Font Turları:**
- **sans-serif:** Modern, toza (Arial, Helvetica, Verdana)
- **serif:** Klassik, rasmi (Georgia, Times New Roman)
- **monospace:** Kod va terminal (Courier, Consolas)
- **cursive:** Qo'lbola yozuv
- **fantasy:** Dekorativ

## 2️⃣ Font O'lchovlari (Font-size)

```css

  font-size: 16px; /* Base size */
}

body {
  font-size: 1rem;     /* 16px */
}

h1 {
  font-size: 2.5rem;   /* 40px */
}

h2 {
  font-size: 2rem;     /* 32px */
}

h3 {
  font-size: 1.5rem;   /* 24px */
}

p {
  font-size: 1rem;     /* 16px */
}

small {
  font-size: 0.875rem; /* 14px */
}
```

---

## 3️⃣ Font Qalinligi (Font-weight)

```css
body {
  font-weight: 400;       /* normal */
}

strong, b {
  font-weight: 700;       /* bold */
}

h1, h2, h3 {
  font-weight: 600;       /* semi-bold */
}

.light {
  font-weight: 300;       /* light */
}

.extra-bold {
  font-weight: 900;       /* extra-bold */
}
```

**Font-weight Qiymatlari:**
- 100 = Thin (Ingichka)
- 300 = Light (Engillik)
- 400 = Normal (Standart)
- 600 = Semi-bold
- 700 = Bold (Qalin)
- 900 = Extra-bold (Juda qalin)

## 5️⃣ Text Tasviri (Text-decoration)

```css
a {
  text-decoration: none;           /* Chiziq yo'q */
}

u {
  text-decoration: underline;      /* Pastdan chiziq */
}

s {
  text-decoration: line-through;   /* O'rta chiziq */
}

.overline {
  text-decoration: overline;       /* Ustidan chiziq */
}

/* Kombinatsiya */
.fancy {
  text-decoration: underline overline wavy red;
}
```
