# CSS Responsive Dizayn va Animatsiya

Responsive dizayn - veb-sayt turli ekran o'lchamlarida (mobil, planshet, desktop) yaxshi ko'rinishini ta'minlaydi. Animatsiya - element harakatini va o'zgarishini amalga oshiradi.

---

## Responsive Dizayn

### 1️⃣ Viewport Meta Tag

HTML'ning `<head>`'da qo'shilishi kerak:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

### 2️⃣ Media Queries

Turli ekran o'lchamlarida turli CSS'ni ishlatish:

```css
/* Asosiy (Desktop) */
.container {
  width: 1200px;
}

/* Planshet (768px va undan katta) */
@media (max-width: 768px) {
  .container {
    width: 100%;
    padding: 10px;
  }
}

/* Mobil (480px va undan katta) */
@media (max-width: 480px) {
  .container {
    padding: 5px;
  }
}
```

---

### 3️⃣ Breakpoints

Standard breakpoints:

```css
/* Mobil telefon */
@media (max-width: 480px) {
  /* 480px va pastroq */
}

/* Kichik planshet */
@media (max-width: 768px) {
  /* 768px va pastroq */
}

/* Planshet */
@media (max-width: 1024px) {
  /* 1024px va pastroq */
}

/* Desktop */
@media (min-width: 1025px) {
  /* 1025px va yuquriga */
}

/* Katta desktop */
@media (min-width: 1440px) {
  /* 1440px va yuquriga */
}
```

---

### 4️⃣ Fluid Layout

Piksel o'rniga foiz ishlating:

```css
.container {
  width: 100%;        /* Barcha kenglik */
  max-width: 1200px;  /* Maximum kenglik */
  margin: 0 auto;
  padding: 0 20px;
}

.column {
  width: 50%;         /* Yarmasi */
}
```

---

### 5️⃣ Responsive Images

Rasmlar turli ekranlarda to'g'ri ko'rinishi:

```css
img {
  max-width: 100%;
  height: auto;       /* Aspect ratio saqlanadi */
}

/* Picture element */
```

```html
<picture>
  <source media="(max-width: 600px)" srcset="small.jpg">
  <source media="(max-width: 1200px)" srcset="medium.jpg">
  <img src="large.jpg" alt="Rasm">
</picture>
```

## Animatsiya

### 1️⃣ Transition (O'tish)

Bir xogidan boshqasiga silliq o'tish:

```css
button {
  background-color: #2563eb;
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: #1d4ed8;
}
```

**Transition Xususiyatlari:**

```css
.element {
  /* property duration timing-function */
  transition: all 0.3s ease;
}

/* Alohida */
.element {
  transition-property: background-color;
  transition-duration: 0.3s;
  transition-timing-function: ease;
  transition-delay: 0.1s;
}
```

---

### 2️⃣ @keyframes Animatsiya

Keyframes bilan murakkab animatsiya:

```css
@keyframes slide {
  0% {
    transform: translateX(0);
  }
  50% {
    transform: translateX(100px);
  }
  100% {
    transform: translateX(0);
  }
}

.box {
  animation: slide 2s infinite;
}
```

---

### 3️⃣ Animation Xususiyatlari

```css
.element {
  animation-name: slide;              /* Animatsiya nomi */
  animation-duration: 2s;             /* Davomiyligi */
  animation-timing-function: ease;    /* Tezlashi */
  animation-delay: 0s;                /* Kechiksh */
  animation-iteration-count: infinite;/* Takroran */
  animation-direction: normal;        /* Yo'nalishi */
}

/* Qisqacha */
.element {
  animation: slide 2s ease 0s infinite normal;
}
```

---

### 4️⃣ Transform (O'zgartirish)

Elementni o'zgartirish:

```css
.box {
  /* O'ngga siljitish */
  transform: translateX(50px);
}

.box {
  /* Vertikal siljitish */
  transform: translateY(-20px);
}

.box {
  /* Kattalashtirish */
  transform: scale(1.5);
}

.box {
  /* Aylanish */
  transform: rotate(45deg);
}

.box {
  /* Egish */
  transform: skew(10deg);
}

/* Kombinatsiya */
.box {
  transform: translateX(50px) rotate(45deg) scale(1.2);
}
```

```html
<!DOCTYPE html>
<html lang="uz">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive va Animatsiya</title>

            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        body {
            font-family: Arial, sans-serif;
            background-color: #f5f5f5;
        }

            width: 100%;
            max-width: 1200px;
            margin: 0 auto;
            padding: 20px;

            padding: 20px;

            margin-bottom: 30px;

        }

        @keyframes slideDown {
            from {
                opacity: 0;
                transform: translateY(-30px);
            }
            to {
                opacity: 1;
                transform: translateY(0);
            }
        }

        h1 {
            font-size: 2rem;
        }

        h2 {
            margin: 30px 0 20px;
            color: #333;
        }

        /* Responsive grid */
        .grid {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            gap: 20px;
            margin-bottom: 30px;
        }

        .card {
            background-color: white;
            border-radius: 8px;
            padding: 20px;
            box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
            transition: all 0.3s ease;
            animation: fadeIn 0.6s ease;
        }

        .card:hover {
            transform: translateY(-10px);
            box-shadow: 0 10px 20px rgba(0, 0, 0, 0.2);
        }

        @keyframes fadeIn {
            from {
                opacity: 0;
                transform: translateY(20px);
            }
            to {
                opacity: 1;
                transform: translateY(0);
            }
        }

        .card h3 {
            color: #2563eb;
            margin-bottom: 10px;
        }

        /* Button animatsiyasi */
        button {
            background-color: #10b981;
            color: white;
            border: none;
            padding: 10px 20px;
            border-radius: 4px;
            cursor: pointer;
            transition: all 0.3s ease;
            font-size: 1rem;
        }

        button:hover {
            background-color: #059669;
            transform: scale(1.05);
        }

        button:active {
            transform: scale(0.95);
        }

        /* Animation examples */
        .spinning {
            width: 50px;
            height: 50px;
            background-color: #2563eb;
            border-radius: 50%;
            animation: spin 2s linear infinite;
            margin: 20px auto;
        }

        @keyframes spin {
            0% {
                transform: rotate(0deg);
            }
            100% {
                transform: rotate(360deg);
            }
        }

        .pulse {
            width: 100px;
            height: 100px;
            background-color: #ef4444;
            border-radius: 50%;
            animation: pulse 1.5s ease-in-out infinite;
            margin: 20px auto;
        }

        @keyframes pulse {
            0%, 100% {
                transform: scale(1);
                opacity: 1;
            }
            50% {
                transform: scale(1.1);
                opacity: 0.7;
            }
        }

        /* Responsive */
        @media (max-width: 1024px) {
            .grid {
                grid-template-columns: repeat(2, 1fr);
            }

            h1 {
                font-size: 1.5rem;
            }
        }

        @media (max-width: 768px) {
            .grid {
                grid-template-columns: 1fr;
            }

            .container {
                padding: 10px;
            }

            button {
                width: 100%;
            }
        }

        @media (max-width: 480px) {
            h1 {
                font-size: 1.2rem;
            }

            h2 {
                font-size: 1rem;
            }

            .card {
                padding: 15px;
            }
        }
    </style>
</head>
<body>

        <header>
            <h1>📱 Responsive Dizayn va Animatsiya</h1>

            <div class="card">
                <h3>Karta 1</h3>
                <p>Mobil ekranda 1 ta ustun.</p>
                <button>Bosing!</button>
            </div>
            <div class="card">
                <h3>Karta 2</h3>
                <p>Planshetda 2 ta ustun.</p>
                <button>Bosing!</button>
            </div>
            <div class="card">
                <h3>Karta 3</h3>
                <p>Desktopda 3 ta ustun.</p>
                <button>Bosing!</button>
            </div>
        </div>

        <h2>Animatsiyalar</h2>
```

## Responsive Design Cheat Sheet

| Breakpoint | O'lchami | Qurilma |
|-----------|----------|---------|
| `< 480px` | Mobil telefon | iPhone, Android |
| `480px - 768px` | Kichik planshet | iPad Mini |
| `768px - 1024px` | Planshet | iPad Pro |
| `> 1024px` | Desktop | Kompyuter |
| `> 1440px` | Katta desktop | Katta monitor |

---

## Animation Timing Functions

| Funksiya | Tavsifi |
|---------|---------|
| `linear` | Bir xil tezlik |
| `ease` | Sekin bosh, tezroq, sekin oxiri |
| `ease-in` | Sekin bosh |
| `ease-out` | Sekin oxiri |
| `ease-in-out` | Sekin bosh va oxiri |
| `cubic-bezier()` | Custom |

## 🎯 Responsive va Animatsiya Qisqacha

- **Responsive:** Turli ekranlarda yaxshi ko'rinish
- **Media queries:** `@media (max-width: 768px) { ... }`
- **Breakpoints:** 480px, 768px, 1024px
- **Transition:** `transition: all 0.3s ease;`
- **Animation:** `@keyframes` + `animation` xususiyati
- **Transform:** `scale()`, `rotate()`, `translateX()`, `skew()`
- **TAVSIYA:** Mobile-first approach - kichik ekrandan boshlash!
