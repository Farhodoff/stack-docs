   # Express.js nima?
   Express.js — bu Node.js uchun minimal va moslashuvchan web ilovalar freymvorki. 
   
   ## Birinchi Express serverimizni yozamiz
   
   <CodeBlock language="javascript" title="index.js">
   ```javascript
   const express = require('express');
   const app = express();
   const port = 3000;

   app.get('/', (req, res) => {
     res.send('Salom Dunyo!');
   });

   app.listen(port, () => {
     console.log(`Server ${port}-portda ishga tushdi`);
   });
   ```
   </CodeBlock>
   
   <Callout type="info" title="Eslab qoling">
   Express.js Node.js ning o'rnatilgan "http" modulining ustiga qurilgan qulay wrapper hisoblanadi.
   </Callout>
