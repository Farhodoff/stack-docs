# CSS - Media Queries va Breakpoint'lar

Media Query - ekran o'lchami, yo'nalishi yoki boshqa xususiyatlari asosida turli CSS qoidalarini qo'llash imkoniyati beradi. Bu responsive dizaynning asosiy texnikasi.

---

## 1️⃣ Media Query Asoslari

Media query qoidasi:

```css
@media (condition) {
  /* Shartga mos CSS */
}

/* Misol */
@media (min-width: 768px) {
  .container {
    width: 750px;
  }
}
```

---

## 2️⃣ Width Breakpoint'lari

Ekran kengligiga mos CSS:

```css
/* Mobil - 320px dan 479px gacha */
@media (max-width: 479px) {
  .layout {
    display: block;
  }
}

/* Small Mobile - 480px dan 639px gacha */
@media (min-width: 480px) and (max-width: 639px) {
  .layout {
    display: flex;
  }
}

/* Tablet - 640px dan 1023px gacha */
@media (min-width: 640px) {
  .layout {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Desktop - 1024px va yuqori */
@media (min-width: 1024px) {
  .layout {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Large Desktop - 1440px va yuqori */
@media (min-width: 1440px) {
  .layout {
    grid-template-columns: repeat(4, 1fr);
  }
}
```

---

## 3️⃣ Height Breakpoint'lari

Ekran balandligiga mos CSS:

```css
/* Kichik balandlik */
@media (max-height: 500px) {
  .navbar {
    padding: 5px 10px;
  }
}

/* Normal balandlik */
@media (min-height: 500px) {
  .navbar {
    padding: 15px 20px;
  }
}

/* Katta balandlik */
@media (min-height: 800px) {
  .sidebar {
    height: 100vh;
    overflow-y: auto;
  }
}
```

---

## 4️⃣ Orientation (Yo'nalishi)

Telefonning vertikal yoki gorizontal yo'nalishi:

```css
/* Portrait - vertikal */
@media (orientation: portrait) {
  .container {
    width: 100%;
    padding: 20px;
  }
}

/* Landscape - gorizontal */
@media (orientation: landscape) {
  .container {
    width: 100vw;
    height: 100vh;
  }
}
```

---

## 5️⃣ Device Pixel Ratio (Retina)

High-DPI ekranlari (Retina) uchun:

```css
/* Standard screens (1x) */
.logo {
  background-image: url('logo.png');
}

/* Retina screens (2x) */
@media (-webkit-min-device-pixel-ratio: 2),
       (min-resolution: 192dpi) {
  .logo {
    background-image: url('logo@2x.png');
    background-size: 50%;
  }
}

/* 3x DPI (Ultra high) */
@media (-webkit-min-device-pixel-ratio: 3),
       (min-resolution: 288dpi) {
  .logo {
    background-image: url('logo@3x.png');
    background-size: 50%;
  }
}
```

## 6️⃣ Color Scheme (Rangni sxema)

Dark mode va light mode:

```css
/* Light mode (default) */
body {
  background-color: white;
  color: #333;
}

/* Dark mode */
@media (prefers-color-scheme: dark) {
  body {
    background-color: #1a1a1a;
    color: #e0e0e0;
  }

  a {
    color: #64b5f6;
  }

  .card {
    background-color: #2a2a2a;
    border-color: #444;
  }
}
```

## 7️⃣ Reduced Motion (Animatsiya uchun)

Foydalanuvchi animatsiyalarni o'chirsa:

```css
/* Default - animatsiyalar ishlasin */
.button {
  transition: all 0.3s ease;
  transform: scale(1);
}

.button:hover {
  transform: scale(1.1);
}

/* Reduced motion - animatsiya o'chirilgan */
@media (prefers-reduced-motion: reduce) {
  .button {
    transition: none;
  }

  .button:hover {
    transform: none;
  }

  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
  }
}
```

---

## 8️⃣ Pointer (Touch yoki Mouse)

Touch qurilmalar yoki mouse:

```css
/* Mouse bilan */
@media (hover: hover) and (pointer: fine) {
  button {
    padding: 8px 16px;
  }

  button:hover {
    background-color: #1d4ed8;
  }
}

/* Touch bilan (hover yo'q) */
@media (hover: none) and (pointer: coarse) {
  button {
    padding: 16px 24px;  /* Kattalashtirish */
    margin: 10px;         /* Ko'proq joy */
  }

  button:active {
    background-color: #1d4ed8;
  }
}
```

---

## 9️⃣ Print Styles

Bosilganda ko'rinishi:

```css
/* Print style */
@media print {
  /* Header va footer yo'q */
  header, footer {
    display: none;
  }

  /* Sidebar yo'q */
  .sidebar {
    display: none;
  }

  /* Kontentni kenglashtirish */
  .main {
    width: 100%;
  }

  /* Link'lar ko'rsatish */
  a::after {
    content: ' (' attr(href) ')';
  }

  /* Sahifa kesimi */
  .page-break {
    page-break-after: always;
  }

  /* Rang tayolashdan saqlash */
  .important {
    border: 1px solid black !important;
    color: black !important;
  }
}
```

## 🔟 Kombinatsiyalangan Media Queries

Bir nechta shartlar:

```css
/* Desktop va landscape */
@media (min-width: 1024px) and (orientation: landscape) {
  .layout {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Mobile yoki touch */
@media (max-width: 768px) or (pointer: coarse) {
  button {
    padding: 15px 20px;
  }
}

/* Desktop va mouse bilan */
@media (min-width: 1024px) and (hover: hover) {
  .menu-item:hover {
    background-color: #f0f0f0;
  }
}

/* Print yoki dark mode */
@media print, (prefers-color-scheme: dark) {
  body {
    color: #000;
    background-color: #fff;
  }
}
```

## 1️⃣2️⃣ Media Query Cheat Sheet

| Shart | Misol | Maqsadi |
|-------|-------|---------|
| `min-width` | `@media (min-width: 768px)` | 768px va kattalashtirish |
| `max-width` | `@media (max-width: 767px)` | 767px va kichiklashtirish |
| `orientation` | `@media (orientation: portrait)` | Yo'nalishi |
| `prefers-color-scheme` | `@media (prefers-color-scheme: dark)` | Dark mode |
| `prefers-reduced-motion` | `@media (prefers-reduced-motion: reduce)` | Animatsiya o'chirish |
| `hover` | `@media (hover: hover)` | Mouse ko'tarilishi |
| `pointer` | `@media (pointer: coarse)` | Touch qurilmalar |
| `min-resolution` | `@media (min-resolution: 192dpi)` | Retina ekranlar |

## 1️⃣3️⃣ Best Practices

```css
/* ✅ YAXSHI - Mobile-first */
.container {
  padding: 20px;
}

@media (min-width: 768px) {
  .container {
    padding: 30px;
  }
}

/* ❌ YOMON - Desktop-first */
.container {
  padding: 30px;
}

@media (max-width: 767px) {
  .container {
    padding: 20px;
  }
}

/* ✅ YAXSHI - Logical names */
.button:hover { }
@media (hover: none) { ... }

/* ✅ YAXSHI - Min-width breakpoints */
@media (min-width: 768px) { ... }

/* ❌ YOMON - Many breakpoints */
@media (max-width: 479px) { ... }
@media (max-width: 767px) { ... }
@media (max-width: 1023px) { ... }
/* Ko'p breakpoint'lar - qo'ng'iroq */
```

---

## 🎯 Media Queries Qisqacha

- `@media (condition)` → Shartga mos CSS
- `min-width` / `max-width` → Ekran kengligiga
- `orientation` → Vertikal yoki gorizontal
- `prefers-color-scheme` → Dark yoki light mode
- `prefers-reduced-motion` → Accessibility uchun
- **TAVSIYA:** Har doim mobile-first boshlang'ich!
