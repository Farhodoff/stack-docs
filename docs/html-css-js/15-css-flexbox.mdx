# CSS Layout - Flexbox

**Flexbox** - zamonaviy joylashtirishning eng oson va kuchli usuli. Elementlarni bir qatorga yoki bir ustuniga joylashtirishni osonlashtiradi.

---

## Flexbox Nima?

Flexbox sizga ruxsat beradi:
- ✅ Elementlarni egri satr yoki ustun bo'ylab joylashtirill
- ✅ Elementlarni markazga joylashtirill
- ✅ Bo'sh joyni teng taqsimla
- ✅ Responsive dizayn yaratish
- ✅ Tartibi bilan oynashish

---

## 1️⃣ Display: Flex

Flexbox'ni aktivlashtirish uchun:

```css
.container {
  display: flex;
}
```

---

## 2️⃣ Flex Direction (Yo'nalishi)

Elementlarni qanday yo'nalishda joylashtirill:

```css
.container {
  display: flex;

  /* Default - qatorga */
  flex-direction: row;
}

/* Ustuniga */
.container {
  flex-direction: column;
}

/* Teskari qatorga */
.container {
  flex-direction: row-reverse;
}

/* Teskari ustuniga */
.container {
  flex-direction: column-reverse;
}
```

---

## 3️⃣ Justify Content (Asosiy Joylashtirish)

Horizontal tekislashturuvi:

```css
.container {
  display: flex;

  /* Default - chapiga */
  justify-content: flex-start;
}

/* O'ngiga */
.container {
  justify-content: flex-end;
}

/* Markazga */
.container {
  justify-content: center;
}

/* Aralarda bo'sh joy */
.container {
  justify-content: space-around;
}

/* Oralig'iga */
.container {
  justify-content: space-between;
}

/* Teng oralig'i bo'lgan */
.container {
  justify-content: space-evenly;
}
```

---

## 4️⃣ Align Items (Vertikal Joylashtirish)

Vertikal tekislashturuvi:

```css
.container {
  display: flex;
  height: 300px;

  /* Ustuniga */
  align-items: flex-start;
}

/* Pastiga */
.container {
  align-items: flex-end;
}

/* Markazga */
.container {
  align-items: center;
}

/* Stretch - to'liq balandlik */
.container {
  align-items: stretch;
}

/* Baseline - basis chiziqiga */
.container {
  align-items: baseline;
}
```

---

## 5️⃣ Wrap (O'ramlash)

Elementlarni qo'shni satrda joylashtirilsa, yangi satrga o'tiss:

```css
.container {
  display: flex;

  /* Default - bitta satrda */
  flex-wrap: nowrap;
}

/* Bir nechta satrlar */
.container {
  flex-wrap: wrap;
}

/* Teskari o'rab */
.container {
  flex-wrap: wrap-reverse;
}
```

---

## 6️⃣ Gap (Bo'sh Joy)

Elementlar orasida bo'sh joy:

```css
.container {
  display: flex;

  /* Horizontal va vertical */
  gap: 20px;
}

/* Alohida */
.container {
  row-gap: 20px;      /* Satrlar orasida */
  column-gap: 15px;   /* Ustunlar orasida */
}
```

---

## 7️⃣ Flex Grow (O'sshni)

Elementlar qancha o'sshini boshqarish:

```css
.item {
  flex-grow: 0;    /* Default - o'shmaydi */
}

.item.grow {
  flex-grow: 1;    /* Bo'sh joyni egallaydi */
}

.item.grow-more {
  flex-grow: 2;    /* Boshqalardan 2 marta ko'p */
}
```

---

## 8️⃣ Flex Shrink (Kichiklashtirish)

Elementlar qancha kichiklagishi mumkin:

```css
.item {
  flex-shrink: 1;  /* Default - kichiklagishi mumkin */
}

.item.no-shrink {
  flex-shrink: 0;  /* Kichiklamaydi */
}
```

---

## 9️⃣ Flex Basis (Asosiy O'lchami)

Elementning asosiy o'lchami:

```css
.item {
  flex-basis: auto;     /* Content o'lchamiga */
}

.item {
  flex-basis: 200px;    /* Tik 200px */
}
```

---

## Flex (Qisqacha)

Barcha flex xususiyatlari bir qatorda:

```css
.item {
  /* flex: grow shrink basis */
  flex: 1 0 200px;
}

/* Teng o'sishi */
.item {
  flex: 1;
}

/* O'sshmasligi */
.item {
  flex: 0 0 200px;
}
```

## Flexbox Cheat Sheet

| Xususiyat | Qiymat | Maqsadi |
|-----------|--------|---------|
| `display` | `flex` | Flexbox'ni aktivlashtirish |
| `flex-direction` | `row`, `column` | Yo'nalishi |
| `justify-content` | `center`, `space-between` | Asosiy joylashtirish |
| `align-items` | `center`, `flex-start` | Vertikal joylashtirish |
| `flex-wrap` | `wrap`, `nowrap` | O'ramlash |
| `gap` | `20px` | Bo'sh joy |
| `flex` | `1 0 200px` | Grow shrink basis |

---

## 🎯 Flexbox Qisqacha

- `display: flex` → Flexbox faolashuvi
- `justify-content` → Horizontal tekislashturuvi
- `align-items` → Vertikal tekislashturuvi
- `gap` → Elementlar orasida bo'sh joy
- `flex: 1` → O'sshni va kichiklagishi
- **TAVSIYA:** Responsive dizayn uchun Flexbox eng yaxshi!
