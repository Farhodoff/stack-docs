# Forma Input Turlari va Validatsiya

Bu darsda siz **barcha HTML5 input turlarini** va **form validatsiyasi**ni chuqur o'rganasiz. Validatsiya **xato ma'lumotlarni oldini oladi** va **foydalanuvchi tajribasi** yaxshilaydi.

---

## 1️⃣ Barcha Input Turları

### Matn va Parol Inputlari

<CodeBlock language="html" title="Matn va Parol">
```html
<!-- Oddiy matn -->
<label for="name">Ismingiz:</label>
<input type="text" id="name" name="name" placeholder="Ismingizni kiriting">

<!-- Parol (matn yoʻqlangan) -->
<label for="pass">Parol:</label>
<input type="password" id="pass" name="pass" placeholder="Parolni kiriting">

<!-- Ko'pqatorli matn -->
<label for="message">Xabar:</label>
<textarea id="message" name="message" rows="5"></textarea>
```
</CodeBlock>

### Email, URL, Telefon

```html
<!-- Email - Avtomatik email validatsiyasi -->
<label for="email">Email:</label>
<input type="email" id="email" name="email" required>

<!-- URL - URL format validatsiyasi -->
<label for="website">Veb saytingiz:</label>
<input type="url" id="website" name="website">

<!-- Telefon raqami -->
<label for="phone">Telefon:</label>
<input type="tel" id="phone" name="phone" pattern="[0-9]{10}">
```

### Raqamlar va Qator

```html
<!-- Raqam -->
<label for="age">Yosh:</label>
<input type="number" id="age" name="age" min="0" max="150">

<!-- Qator (slider) -->
<label for="volume">Tovush:</label>
<input type="range" id="volume" name="volume" min="0" max="100">
```

### Sana va Vaqt

```html
<!-- Sana -->
<label for="birthdate">Tug'ilish sanasi:</label>
<input type="date" id="birthdate" name="birthdate">

<!-- Sana va vaqt -->
<label for="meeting">Uchrashuvni tanlang:</label>
<input type="datetime-local" id="meeting" name="meeting">

<!-- Vaqt -->
<label for="time">Vaqti:</label>
<input type="time" id="time" name="time">

<!-- Oy -->
<label for="month">Oy:</label>
<input type="month" id="month" name="month">

<!-- Hafta -->
<label for="week">Hafta:</label>
<input type="week" id="week" name="week">
```

### Rang va Fayl

```html
<!-- Rang tanlash -->
<label for="color">Sevimli rang:</label>
<input type="color" id="color" name="color" value="#0066ff">

<!-- Fayl yuklash -->
<label for="document">Fayl yuklang:</label>
<input type="file" id="document" name="document" accept=".pdf,.doc,.docx">

<!-- Bir nechta fayl -->
<label for="files">Fayllar:</label>
<input type="file" id="files" name="files" multiple>
```

### Checkbox va Radio

### Select va Textarea

```html
<!-- Select - Tushish menyu -->
<label for="country">Davlat:</label>
```

## 2️⃣ Validatsiya Atributlari

### `required` - Majburiy Maydon

```html
<!-- Ushbu maydon to'ldirilishi majburiy -->
<input type="text" name="username" required>
<input type="email" name="email" required>

<!-- Checkbox majburiy -->
<input type="checkbox" name="terms" required>
<label for="terms">Shartlarni qabul qilaman</label>
```

### `minlength` va `maxlength` - Uzunlik Cheklash

```html
<!-- Kamida 3 ta belgi -->
<input type="text" name="username" minlength="3">

<!-- Maksimum 50 ta belgi -->
<input type="text" name="title" maxlength="50">

<!-- Ikkalasini birlashtirish -->
<input type="password" name="pass" minlength="8" maxlength="20">
```

### `min` va `max` - Raqamlarni Cheklash

```html
<!-- 0 dan 150 gacha -->
<input type="number" name="age" min="0" max="150">

<!-- Sana cheklash -->
<input type="date" name="birthdate"
       min="1950-01-01"
       max="2010-12-31">

<!-- Vaqt cheklash -->
<input type="time" name="worktime"
       min="08:00"
       max="17:00">
```

### `step` - Qadamni Belgilash

```html
<!-- 0.1 qadamda (0, 0.1, 0.2, ...) -->
<input type="number" name="price" step="0.01">

<!-- Toq raqamlargina (1, 3, 5, 7, ...) -->
<input type="number" name="odd" step="2">

<!-- Sana - haftalar bo'ylab -->
<input type="date" name="date" step="7">
```

### `pattern` - Regular Expression Validatsiyasi

```html
<!-- Faqat 3 ta harfli kod -->
<input type="text" name="code" pattern="[A-Z]{3}">

<!-- Telefon raqami format: XXX-XXX-XXXX -->
<input type="tel" name="phone" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}">

<!-- Lotin harflari va raqamlari -->
<input type="text" name="username" pattern="[a-zA-Z0-9_]+" title="Faqat harflar, raqamlar va underscore">

<!-- O'zbek telefon raqami: +998901234567 -->
<input type="tel" name="phone" pattern="\+998[0-9]{9}">
```

## 3️⃣ Keyingi Input va Placeholder

### `placeholder` - Maslahat Matnini

```html
<!-- Placeholder - input bo'sh bo'lganida ko'rinadi -->
<input type="email" placeholder="your-email@example.com">
<textarea placeholder="Xabaringizni yozing..."></textarea>

<!-- Placeholder matn oddiy tavsifdir, label emas! -->
<label for="email">Email Manzili:</label>
<input type="email" id="email" placeholder="example@gmail.com">
```

### `list` va `<datalist>` - Avto-To'ldirish

```html
<!-- Input bilan avto-to'ldirish -->
<input type="text" name="country" list="countries" placeholder="Davlatni yozing...">

<datalist id="countries">
    <option value="O'zbekiston">
    <option value="Qozog'iston">
    <option value="Tojikiston">
    <option value="Turkmaniston">
    <option value="Qirgiziston">
</datalist>
```

## ✅ Best Practices

### 1. Har Doim Validatsiya Qo'shing

```html
<!-- ❌ Yomon -->
<input type="email">

<!-- ✅ Yaxshi -->
<input type="email" required>
```

### 2. Pattern Bilan Maslahat

```html
<!-- ✅ Yaxshi - pattern o'zining maslahat beradi -->
<input type="text" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}"
       title="Telefon formati: XXX-XXX-XXXX">
```

### 3. Min/Max Cheklash

```html
<!-- ✅ Yaxshi -->
<input type="number" name="age" min="18" max="100">
<input type="date" name="birthdate" max="2006-12-31">
```

### 4. Listdan Foydalanish

```html
<!-- ✅ Yaxshi - avto-to'ldirish -->
<input type="text" list="countries">
<datalist id="countries">
    <option value="O'zbekiston">
    <option value="Qozog'iston">
</datalist>
```

## 🖼️ Rasmlar (`<img>` Tegi)

**`<img>` tegi** - veb-sahifada rasmlar ko'rsatish uchun.

### Asosiy Rasm Struktura

<CodeBlock language="html" title="Rasm Misoli">
```html
<img src="photo.jpg" alt="Tavsif matni" width="300" height="200">
```
</CodeBlock>

### Rasm Atributlari

| Atribut | Ma'nosi | Misol |
|---------|---------|-------|
| `src` | Rasm yo'li (MAJBURIY) | `src="image.jpg"` |
| `alt` | Rasm yuklanmasa yoki screen reader uchun | `alt="Rasm tavsifi"` |
| `width` | Kenglik | `width="300"` |
| `height` | Balandlik | `height="200"` |
| `title` | Tooltip matn | `title="Rasmga hover..."` |

### Rasm Yo'llari

#### Absolyut Yo'l

```html
<!-- Tashqi saytdan -->
<img src="https://example.com/image.jpg" alt="Tashqi rasm">

<!-- Proyekt ichidagi root'dan -->
<img src="/images/logo.png" alt="Logotip">
```

#### Nisbiy Yo'l

```html
<!-- Joriy papkada -->
<img src="photo.jpg" alt="Rasm">

<!-- Ata papkada -->
<img src="../images/photo.jpg" alt="Rasm">
```

### Responsive Rasm

```html
<!-- Width va height nisbati -->
<img src="image.jpg" alt="Rasm" width="400" height="300">

<!-- Responsive (CSS bilan) -->
<img src="image.jpg" alt="Rasm" style="max-width: 100%; height: auto;">
```

### Picture (Turli Rasmlar, Turli Ekranlar Uchun)

```html
<picture>
    <!-- Malenkiy ekran -->
    <source media="(max-width: 480px)" srcset="image-small.jpg">
    
    <!-- O'rtacha ekran -->
    <source media="(max-width: 768px)" srcset="image-medium.jpg">
    
    <!-- Katta ekran -->
    <img src="image-large.jpg" alt="Responsive Rasm">
</picture>
```

## 📝 Ro'yxatlar

HTML'da 3 xil ro'yxat turi bor: tartibsiz, tartibli, va izohli.

### 1️⃣ Tartibsiz Ro'yxat (`<ul>`)

Tartibi muhim emas. Har bir element bosh nuqta bilan ko'rinadi:

<CodeBlock language="html" title="Tartibsiz Ro'yxat">
```html
<ul>
    <li>Olma</li>
    <li>Banan</li>
    <li>Gilos</li>
    <li>Anor</li>
</ul>
```
</CodeBlock>

**Ko'rinadi:**
- Olma
- Banan
- Gilos
- Anor

### 2️⃣ Tartibli Ro'yxat (`<ol>`)

Tartibi muhim. Har bir element raqam bilan ko'rinadi:

<CodeBlock language="html" title="Tartibli Ro'yxat">
```html
<ol>
    <li>Birinchi qadam</li>
    <li>Ikkinchi qadam</li>
    <li>Uchinchi qadam</li>
</ol>
```
</CodeBlock>

**Ko'rinadi:**
1. Birinchi qadam
2. Ikkinchi qadam
3. Uchinchi qadam

### 3️⃣ Izohli Ro'yxat (`<dl>`)

Shartlar va ularning ta'riflari. `<dt>` - shart, `<dd>` - ta'rif:

<CodeBlock language="html" title="Izohli Ro'yxat">
```html
<dl>
    <dt>HTML</dt>
    <dd>Veb-sahifaning tuzilishi va kontenti</dd>
    
    <dt>CSS</dt>
    <dd>Veb-sahifaning ko'rinishi va stilizatsiya</dd>
    
    <dt>JavaScript</dt>
    <dd>Veb-sahifaning interaktivligi va dinamikasi</dd>
</dl>
```
</CodeBlock>
