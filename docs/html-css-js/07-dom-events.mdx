# Formalar - Asoslar

Formalar **foydalanuvchidan ma'lumot olish** uchun HTML'dagi eng muhim elementlardir. Bu darsda siz form strukturasini, input turlarini va ma'lumot yuborishni o'rganasiz.

---

## 1️⃣ Form Tegi

`<form>` tegi ma'lumot olish va yuborish uchun ishlatiladi.

### Asosiy Form Struktura

<CodeBlock language="html" title="Oddiy Form">
```html
<form action="/submit" method="POST">
    <!-- Form elementlari shu yerda -->
    <label for="ism">Ismingiz:</label>
    <input type="text" id="ism" name="ism">

    <button type="submit">Yuborish</button>
</form>
```
</CodeBlock>

### Form Atributlari

#### `action` - Ma'lumot Qayerga Yuboriladi

```html
<!-- Server adresiga yuborish -->
<form action="/api/register" method="POST">
    <!-- Forma elementlari -->
</form>

<!-- Email yuborish (masalan, newsletter) -->
<form action="mailto:info@example.com" method="POST" enctype="text/plain">
    <!-- Forma elementlari -->
</form>
```

#### `method` - HTTP Metodi

| Metodi | Foydalanishi | Ma'lumotlar |
|--------|-----------|-----------|
| `GET` | Qidiruv, filtr | URL ichida ko'rinadi (xavfli emas) |
| `POST` | Registratsiya, login | Body ichida (xavfsizroq) |

```html
<!-- GET metodi - URL ichida ko'rinadi -->
<form action="/search" method="GET">
    <input type="text" name="q">
    <button>Qidirish</button>
</form>
<!-- URL: /search?q=html&type=html -->

<!-- POST metodi - Body ichida (yashirun) -->
<form action="/register" method="POST">
    <input type="text" name="username">
    <button>Ro'yxatdan o'tish</button>
</form>
```

#### `enctype` - Ma'lumot Kodlanish Turi

```html
<!-- Default - Matn ma'lumotlari -->
<form action="/submit" method="POST" enctype="application/x-www-form-urlencoded">

<!-- Fayl yuklash uchun -->
<form action="/upload" method="POST" enctype="multipart/form-data">
    <input type="file" name="file">
</form>
```

## 2️⃣ Label va Input

`<label>` va `<input>` formaning asosiy qismidir.

### <label> Tegi

`<label>` input'ning nomlari belgilaydi va accessibility'ni yaxshilaydi:

```html
<!-- ❌ Yomon - label yo'q -->
<input type="text">

<!-- ✅ Yaxshi - label bor -->
<label for="email">Email manzili:</label>
<input type="email" id="email" name="email">
```

### <input> Tegi

```html
<!-- Matn kiritish -->
<input type="text" name="username">

<!-- Email -->
<input type="email" name="email">

<!-- Parol -->
<input type="password" name="password">

<!-- Raqam -->
<input type="number" name="age" min="1" max="120">

<!-- Sana -->
<input type="date" name="birthdate">

<!-- Checkbox -->
<input type="checkbox" name="agree" id="agree">
<label for="agree">Shartlarni qabul qilaman</label>

<!-- Radio button -->
<input type="radio" name="gender" value="male" id="male">
<label for="male">Erkak</label>

<!-- Fayl yuklash -->
<input type="file" name="document">

<!-- Submit tugma -->
<button type="submit">Yuborish</button>
```

## 3️⃣ Input Atributlari

### `name` - Forma Ma'lumoti Uchun Identifikator

```html
<!-- name mavjud bo'lsa, server ma'lumot oladi -->
<input type="text" name="username">

<!-- name yo'q bo'lsa, server ma'lumot olmaydi -->
<input type="text">
```

### `placeholder` - Maslahat Matnini

```html
<input type="email" placeholder="your-email@example.com">
<textarea placeholder="Xabaringizni yozing..."></textarea>
```

### `required` - Majburiy Maydon

```html
<!-- Ushbu maydon to'ldirilishi majburiy -->
<input type="text" name="username" required>

<!-- Shunga oyna yoq bo'lsa, forma yuborilmaydi -->
<form onsubmit="return checkForm()">
    <input type="email" name="email" required>
    <button type="submit">Yuborish</button>
</form>
```

### `disabled` va `readonly`

```html
<!-- disabled - o'zgartirilmaydi va yuborilmaydi -->
<input type="text" value="O'zgartirib bo'lmaydi" disabled>

<!-- readonly - o'zgartirilmaydi, lekin yuboriladi -->
<input type="text" value="Ko'rish uchun faqat" readonly>
```

### `value` - Qolip Qiymat

```html
<input type="text" name="city" value="Toshkent">
<textarea name="bio">Mening bio matnlari</textarea>
```

## ✅ Best Practices

### 1. Har Doim Label Ishlatish

```html
<!-- ❌ Yomon - label yo'q -->
<input type="email">

<!-- ✅ Yaxshi - label bor -->
<label for="email">Email:</label>
<input type="email" id="email" name="email">
```

### 2. `name` Atributini Beya

```html
<!-- ❌ Yomon - name yo'q -->
<input type="text">

<!-- ✅ Yaxshi - name bor -->
<input type="text" name="username">
```

### 3. Aniqlashni Ishlatish

```html
<!-- ✅ Yaxshi - aniqlash -->
<form action="/register" method="POST" enctype="multipart/form-data">
    <!-- Fayl yuklash -->
</form>
```

### 4. Placeholder dan Label Joyiga Foydalanmang

```html
<!-- ❌ Yomon - label yo'q -->
<input type="email" placeholder="Enter your email">

<!-- ✅ Yaxshi - label bor -->
<label for="email">Email Manzili:</label>
<input type="email" id="email" name="email" placeholder="example@email.com">
```

## Elementlarni Yaratish va O'chirish

```javascript
// Element yaratish
const newDiv = document.createElement("div");
newDiv.textContent = "Yangi div";
newDiv.className = "container";

// DOM'ga qo'shish
document.body.appendChild(newDiv);

// Specific joyga qo'shish
const parent = document.getElementById("parent");
parent.appendChild(newDiv);

// O'chirish
newDiv.remove();
// yoki
parent.removeChild(newDiv);
```

## Events va Event Listeners

```javascript
const button = document.querySelector("button");

// Click event
button.addEventListener("click", () => {
  console.log("Tugma bosildi!");
});

// Input event
const input = document.querySelector("input");
input.addEventListener("input", (event) => {
  console.log("Matn:", event.target.value);
});

// Form submit
const form = document.querySelector("form");
form.addEventListener("submit", (event) => {
  event.preventDefault(); // Default sarmayish o'chirish
  console.log("Forma submited!");
});

// Mouse events
element.addEventListener("mouseover", () => {
  console.log("Mouse hover");
});

element.addEventListener("mouseout", () => {
  console.log("Mouse leave");
});
```

## Praktik Misol - Todo List

```html
<div id="app">
  <input id="todoInput" type="text" placeholder="Qo'shing">
  <button id="addBtn">Qo'shish</button>
  <ul id="todoList"></ul>
</div>
```

```javascript
const input = document.getElementById("todoInput");
const btn = document.getElementById("addBtn");
const list = document.getElementById("todoList");

btn.addEventListener("click", () => {
  const text = input.value;

  if (text.trim() === "") return;

  // Li element yaratish
  const li = document.createElement("li");
  li.textContent = text;

  // Delete tugmasi
  const deleteBtn = document.createElement("button");
  deleteBtn.textContent = "O'chirish";
  deleteBtn.addEventListener("click", () => {
    li.remove();
  });

  li.appendChild(deleteBtn);
  list.appendChild(li);

  input.value = "";
});
```
