# HTML Jadvallar (Tables)

## 📊 Jadvallar Nima?

**Jadvallar** – ma'lumotlarni satr va ustunlar ko'rinishida (grid formato'da) ko'rsatish uchun ishlatiladi.

Masalan:
- Talabalar ro'yxati
- Narx quymasi
- Grafik yoki holatlar
- Qo'shilgan ma'lumotlar

---

## 🏗️ Jadvalni Tuzilishi

<CodeBlock language="html" title="Asosiy Jadval Struktura">
```html
<table border="1">
    <thead>
        <tr>
            <th>Sarlavha 1</th>
            <th>Sarlavha 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Ma'lumot 1</td>
            <td>Ma'lumot 2</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Juga</td>
            <td>Summа</td>
        </tr>
    </tfoot>
</table>
```
</CodeBlock>

### Jadval Elementlari

| Element | Ma'nosi |
|---------|---------|
| `<table>` | Jadvalni o'ramlash |
| `<thead>` | Sarlavhalar (yuqori) |
| `<tbody>` | Asosiy ma'lumotlar |
| `<tfoot>` | Juga (pastki) |
| `<tr>` | Satr (table row) |
| `<th>` | Sarlavha cell (qalin, markazda) |
| `<td>` | Ma'lumot cell (table data) |

---

## 🎨 Jadval Atributlari

| Atribut | Ma'nosi | Misol |
|---------|---------|-------|
| `border` | Chiziqning qalinligi | `border="1"` |
| `cellpadding` | Cell ichidagi joy | `cellpadding="10"` |
| `cellspacing` | Celllar orasidagi jo'y | `cellspacing="5"` |
| `width` | Jadvallning kengligini | `width="500"` |
| `height` | Jadvallning balandiligini | `height="300"` |

## 🔗 Colspan (Ustunlarni Birlashtirish)

**colspan** – ma'lumotni bir nechta ustunlar orasida cho'zish:

<CodeBlock language="html" title="Colspan Misoli">
```html
<table border="1" cellpadding="8">
    <tr>
        <th colspan="3">Eng Muhim Ma'lumotlar</th>
        <!-- Bu sarlavha 3 ta ustunni egallaydi -->
    </tr>
    <tr>
        <th>Ism</th>
        <th>Yoshi</th>
        <th>Shahar</th>
    </tr>
    <tr>
        <td>Ali</td>
        <td>25</td>
        <td>Toshkent</td>
    </tr>
    <tr>
        <td colspan="2">Ali Toshkentda yasaydi</td>
        <!-- 2 ta ustunni egallaydi -->
        <td>Toshkent</td>
    </tr>
</table>
```
</CodeBlock>

## 📏 Rowspan (Satrlarni Birlashtirish)

**rowspan** – ma'lumotni bir nechta satrlar orasida cho'zish:

<CodeBlock language="html" title="Rowspan Misoli">
```html
<table border="1" cellpadding="8">
    <tr>
        <th>Ism</th>
        <th>Savdo Mahsulotlari</th>
        <th>Narx</th>
    </tr>
    <tr>
        <td rowspan="3">Ali</td>
        <!-- Ali 3 ta qator ichida ko'rinadi -->
        <td>Olma</td>
        <td>50,000 so'm</td>
    </tr>
    <tr>
        <!-- Ali bu qatorda ham ko'rinadi -->
        <td>Banan</td>
        <td>30,000 so'm</td>
    </tr>
    <tr>
        <!-- Ali bu qatorda ham ko'rinadi -->
        <td>Gilos</td>
        <td>60,000 so'm</td>
    </tr>
</table>
```
</CodeBlock>

## 🎯 Best Practices

1. **`<thead>`, `<tbody>`, `<tfoot>` ishlatish** – Semantik va strukturali
2. **CSS bilan styling qilish** – HTML atributlaridan emas
3. **`border-collapse: collapse`** – Chiziqlar takrorlanmasligi uchun
4. **Responsive tablalar** – Mobil'da horizontal scroll
5. **Accessibility** – Screen reader'lar uchun kabi atributlar

---

## 📱 Responsive Jadval (CSS bilan)

<CodeBlock language="html" title="Responsive Jadval">
```html
<style>
    @media (max-width: 600px) {
        table, thead, tbody, th, td, tr {
            display: block;
        }

        thead tr {
            display: none;
        }

        td {
            padding-left: 50%;
            position: relative;
        }

        td:before {
            content: attr(data-label);
            position: absolute;
            left: 6px;
            font-weight: bold;
        }
    }
</style>

<table>
    <thead>
        <tr>
            <th>Ism</th>
            <th>Yoshi</th>
            <th>Shahar</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td data-label="Ism">Ali</td>
            <td data-label="Yoshi">25</td>
            <td data-label="Shahar">Toshkent</td>
        </tr>
    </tbody>
</table>
```
</CodeBlock>
