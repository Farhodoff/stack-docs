# Video va Audio Teglar

HTML5 `<video>` va `<audio>` teglar multimedia kontent qo'shishni oson qildi. Bu darsda siz video va audio fayllarini HTML'ga qanday qo'shishni va ularni boshqarishni o'rganasiz.

---

## 1️⃣ Video Tegi

`<video>` tegi sahifaga video qo'shadi. Brauzer avtomatik video pleyer bilan ko'rsatadi.

### Asosiy Video Tegi

<CodeBlock language="html" title="Sodda Video">
```html
<!-- Eng asosiy vidyo -->
<video src="/videos/sample.mp4"></video>

<!-- Video kontrollari bilan -->
<video src="/videos/sample.mp4" controls></video>

<!-- Video o'lchamlari bilan -->
<video src="/videos/sample.mp4"
       width="640"
       height="480"
       controls></video>

<!-- Video plakati (ko'rinish) bilan -->
<video src="/videos/sample.mp4"
       poster="/images/poster.jpg"
       controls></video>
```
</CodeBlock>

### Video Tegi Atributlari

#### `controls` - Pleyer Kontrollari

```html
<!-- ❌ Kontrollari yo'q - foydalanuvchi video boshqa tilishtirolmaydi -->
<video src="/video.mp4"></video>

<!-- ✅ Kontrollari bor - play/pause/volume/progress bar -->
<video src="/video.mp4" controls></video>
```

#### `autoplay` - Sahifa Yuklanganda Avtomatik Tasvish

```html
<!-- Video sahifa yuklanganda avtomatik boshlanadi -->
<video src="/video.mp4" autoplay></video>

<!-- Ko'p brauzerlar autoplay dan ijozat olmaydi -->
<!-- Mute bilan autoplay ishlaydi -->
<video src="/video.mp4" autoplay muted></video>
```

#### `muted` - Yaglangan

```html
<!-- Video yaglangan (tovush yo'q) -->
<video src="/video.mp4" muted></video>

<!-- Autoplay + Muted (brauzerda ishlaydi) -->
<video src="/video.mp4" autoplay muted></video>
```

#### `loop` - Qayta Tasvish

```html
<!-- Video tugasa, yana boshlanadi -->
<video src="/video.mp4" controls loop></video>
```

#### `preload` - Oldindan Yuklanish

```html
<!-- Hech narsani oldindan yuklama (default) -->
<video src="/video.mp4" preload="none"></video>

<!-- Metadata oldindan yukla (kadr soni, davomiyligi) -->
<video src="/video.mp4" preload="metadata"></video>

<!-- Butun videoni oldindan yukla -->
<video src="/video.mp4" preload="auto"></video>
```

#### `poster` - Videoning Qoplan Rasmi

Video yuklanmaganda bu rasm ko'rsatiladi:

```html
<video src="/video.mp4"
       poster="/images/thumbnail.jpg"
       controls>
</video>
```

#### `width` va `height` - O'lcham

```html
<!-- Pikselda o'lcham -->
<video src="/video.mp4" width="640" height="480" controls></video>

<!-- Responsive o'lcham (CSS bilan) -->
<video src="/video.mp4" style="width: 100%; height: auto;" controls></video>
```

## 2️⃣ Multiple Video Format'lari

Turli brauzerlar turli format'larni qo'llab-quvvatlaydi. Shuning uchun bir nechta format taqdim etishlari kerak.

<CodeBlock language="html" title="Bir Nechta Format'li Video">
```html
<video width="640" height="480" controls poster="/poster.jpg">
    <!-- MP4 format - Ko'p brauzerlar qo'llab-quvvatlaydi -->
    <source src="/videos/video.mp4" type="video/mp4">

    <!-- WebM format - Firefox, Chrome -->
    <source src="/videos/video.webm" type="video/webm">

    <!-- OGG format - Firefox, Chrome (eski) -->
    <source src="/videos/video.ogv" type="video/ogg">

    <!-- Brauzer hech qaysi formatni qo'llab-quvvatlasa -->
    Afsuski, brauzeringiz video qo'llab-quvvallamaydi.
    <a href="/videos/video.mp4">Videoni yuklab oling</a>
</video>
```
</CodeBlock>

### Video Format'larining Taqqoslash

| Format | MIME Type | Brauzerlar | Afzalliklar |
|--------|-----------|-----------|-----------|
| MP4 | video/mp4 | Barchasi | Keng talab qilish |
| WebM | video/webm | Chrome, Firefox, Edge | Yaxshi sifat, kichik fayl |
| OGG | video/ogg | Firefox, Chrome (eski) | Open source |

**Maslahat:** MP4 + WebM kombinatsiyasi keng talab qiluvchi.

## 3️⃣ Audio Tegi

`<audio>` tegi sahifaga audio fayllarini qo'shadi.

<CodeBlock language="html" title="Audio Misollari">
```html
<!-- Sodda audio -->
<audio src="/audio/music.mp3" controls></audio>

<!-- Audio o'lchamlari bilan -->
<audio src="/audio/music.mp3"
       width="300"
       controls></audio>

<!-- Audio Autoplay + Muted -->
<audio src="/audio/background.mp3"
       autoplay
       muted
       loop></audio>

<!-- Bir nechta format'li audio -->
<audio controls>
    <source src="/audio/music.mp3" type="audio/mpeg">
    <source src="/audio/music.ogg" type="audio/ogg">
    <source src="/audio/music.wav" type="audio/wav">
    Brauzeringiz audio qo'llab-quvvallamaydi.
</audio>

<!-- Preload bilan -->
<audio src="/audio/music.mp3"
       preload="metadata"
       controls></audio>
```
</CodeBlock>

### Audio Format'lari

| Format | MIME Type | Brauzerlar | Afzalliklar |
|--------|-----------|-----------|-----------|
| MP3 | audio/mpeg | Barchasi | Keng talab qilish |
| OGG | audio/ogg | Firefox, Chrome | Open source |
| WAV | audio/wav | Barchasi | Yaxshi sifat |
| AAC | audio/aac | Safari, Chrome | Faqat |

## 4️⃣ YouTube va Boshqa Tashqi Video'lar

YouTube va boshqa platformalardan embed qilsangiz, `<iframe>` ishlatiladi:

<CodeBlock language="html" title="YouTube Video Embed">
```html
<!-- YouTube videoni embed qilish -->
<iframe width="560"
        height="315"
        src="https://www.youtube.com/embed/dQw4w9WgXcQ"
        title="YouTube video player"
        frameborder="0"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
        allowfullscreen>
</iframe>

<!-- Responsive YouTube video -->
<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
    <iframe style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
            src="https://www.youtube.com/embed/dQw4w9WgXcQ"
            frameborder="0"
            allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share"
            allowfullscreen>
    </iframe>
</div>
```
</CodeBlock>

## ✅ Best Practices

### 1. Ko'p Format'lar Taqdim Etish

```html
<!-- ❌ Yomon - bitta format (ba'zi brauzerlar ko'rib olmaydi) -->
<video src="/video.mp4" controls></video>

<!-- ✅ Yaxshi - MP4 + WebM -->
<video controls>
    <source src="/video.mp4" type="video/mp4">
    <source src="/video.webm" type="video/webm">
</video>
```

### 2. Poster Rasmini Ishlatish

```html
<!-- ❌ Yomon - placeholder yo'q -->
<video src="/video.mp4" controls></video>

<!-- ✅ Yaxshi - Qoplan rasmi bor -->
<video src="/video.mp4" poster="/poster.jpg" controls></video>
```

### 3. Autoplay + Muted

```html
<!-- ❌ Yomon - autoplay bir o'zi (ko'p brauzerda ishlama)  -->
<video src="/video.mp4" autoplay></video>

<!-- ✅ Yaxshi - autoplay + muted (ishlaydi) -->
<video src="/video.mp4" autoplay muted></video>
```

### 4. Responsive Multimedia

```html
<!-- ❌ Yomon - fixed o'lcham -->
<video src="/video.mp4" width="800" height="600" controls></video>

<!-- ✅ Yaxshi - responsive -->
<video src="/video.mp4"
       controls
       style="width: 100%; max-width: 800px; height: auto;">
</video>
```

### 5. Fallback Content

```html
<!-- ✅ Yaxshi - brauzer qo'llab-quvvallamasa download link -->
<video controls>
    <source src="/video.mp4" type="video/mp4">
    <source src="/video.webm" type="video/webm">
    <p>Brauzeringiz video qo'llab-quvvallamaydi.
       <a href="/video.mp4">Videoni yuklab oling</a>
    </p>
</video>
```

| Atribut | Vazifasi | Misol |
