# HTML Proyekti - Multi-page Website

Bu darsda siz **3-sahifali to'liq veb-sayt** yaratasiz: **Bosh Sahifa**, **Haqida** va **Kontakt** sahifalari. Bu proyekt barcha HTML darslarida o'rgangan narsalarni birlashtiradi.

---

## 📋 Proyektning Tavsifi

Biz **"Fullstack Docs" Blogging Platformasi** uchun veb-sayt yaratamiz:

- **Bosh Sahifa:** Saytning asosiy sahifasi, navbar, hero qismi
- **Haqida Sahifasi:** Kompaniya haqida ma'lumot, misyon, ko'zlamalar
- **Kontakt Sahifasi:** Kontakt formasi, xabar yuborish

**Texnologiyalar:**
- HTML5 (semantik)
- CSS3 (qolipga qarang)
- Responsive design

---

## 🏗️ Proyektning Fayl Strukturasi

```
saytim/
├── index.html          # Bosh sahifa
├── about.html          # Haqida
├── contact.html        # Kontakt
├── css/
│   └── styles.css      # Umumiy stillar
└── images/
    ├── logo.png
    └── hero-image.jpg
```

## 🔗 CSS Qo'shish: 3 Ta Usul

### 1️⃣ Inline CSS (Tugalroq)

HTML elementi ichiga yozish:

```html
<p style="color: red; font-size: 18px;">Bu qizil matn</p>
```

**Muammolari:**
- ❌ Kod nopok
- ❌ Takrorlanadi
- ❌ Qiyin boshqarish

### 2️⃣ Internal CSS (O'rtacha)

`<head>` ichiga `<style>` teg bilan:

<CodeBlock language="html" title="Internal CSS">
```html
<!DOCTYPE html>
<html>
<head>
    <style>
        h1 { color: blue; }
        p { font-size: 18px; }
        button { background: green; }
    </style>
</head>
<body>
    <h1>Sarlavha</h1>
    <p>Matn</p>
    <button>Tugma</button>
</body>
</html>
```
</CodeBlock>

**Foydalari:**
- ✅ Korxonalashtirilgan
- ❌ Faqat bir sahifada
- ❌ Boshqa fayllar bilan bo'lmaydi

### 3️⃣ External CSS (Eng Yaxshi)

Alohida `.css` fayl:

<CodeBlock language="html" title="index.html">
```html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Sarlavha</h1>
    <p>Matn</p>
    <button>Tugma</button>
</body>
</html>
```
</CodeBlock>

<CodeBlock language="css" title="styles.css">
```css
h1 {
    color: blue;
    font-size: 32px;
}

p {
    font-size: 18px;
    line-height: 1.6;
}

button {
    background: green;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
}
```
</CodeBlock>

**Foydalari:**
- ✅ Barcha sahifalarda ishlaydi
- ✅ Oson boshqarish
- ✅ Kodlar nopak
- ✅ Katta proyektlar uchun ideal
