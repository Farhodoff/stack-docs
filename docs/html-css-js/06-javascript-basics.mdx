# Jadvallar (Tables)

HTML jadvallar **ma'lumotlarni satr va ustunlar ko'rinishida ko'rsatish** uchun ishlatiladi. Bu darsda to'liq jadvalni yaratishni va semantik tuzilmani o'rganasiz.

---

## 1️⃣ Jadvalni Tuzilishi

Jadvalni yaratish uchun eskadir taglardan foydalaniladi:

- `<table>` - Jadvalni belgilaydi
- `<tr>` (Table Row) - Satr
- `<td>` (Table Data) - Ma'lumot katagi
- `<th>` (Table Header) - Sarlavha katagi
- `<thead>` - Jadvalni bosh qismi
- `<tbody>` - Jadvalni asosiy qismi
- `<tfoot>` - Jadvalni pastki qismi

<CodeBlock language="html" title="Asosiy Jadvalni Struktura">
```html
<table border="1">
    <!-- Jadvalni bosh qismi (sarlavhalar) -->
    <thead>
        <tr>
            <th>Sarlavha 1</th>
            <th>Sarlavha 2</th>
            <th>Sarlavha 3</th>
        </tr>
    </thead>

    <!-- Jadvalni asosiy qismi (ma'lumotlar) -->
    <tbody>
        <tr>
            <td>Ma'lumot 1-1</td>
            <td>Ma'lumot 1-2</td>
            <td>Ma'lumot 1-3</td>
        </tr>
        <tr>
            <td>Ma'lumot 2-1</td>
            <td>Ma'lumot 2-2</td>
            <td>Ma'lumot 2-3</td>
        </tr>
    </tbody>

    <!-- Jadvalni pastki qismi (juga, summa) -->
    <tfoot>
        <tr>
            <td colspan="2">Juga</td>
            <td>$1000</td>
        </tr>
    </tfoot>
</table>
```
</CodeBlock>

## 2️⃣ Jadvalni Elementlari

### <th> va <td> Farqi

| Element | Maqsadi | Brauzer Ko'rinishi |
|---------|---------|------------------|
| `<th>` | Sarlavha katagi | **Qalin** va **markazlashgan** |
| `<td>` | Ma'lumot katagi | Oddiy matn |

```html
<!-- Birinchi satr - sarlavhalar -->
<tr>
    <th>Ism</th>
    <th>Familiyasi</th>
    <th>Yoshi</th>
</tr>

<!-- Ikkinchi satr - ma'lumotlar -->
<tr>
    <td>Ali</td>
    <td>Karimov</td>
    <td>25</td>
</tr>
```

### <thead>, <tbody>, <tfoot>

```html
<table>
    <!-- Sarlavhalar -->
    <thead>
        <tr>
            <th>Tovar</th>
            <th>Narxi</th>
        </tr>
    </thead>

    <!-- Ma'lumotlar -->
    <tbody>
        <tr>
            <td>Kitob</td>
            <td>$10</td>
        </tr>
        <tr>
            <td>Qalam</td>
            <td>$5</td>
        </tr>
    </tbody>

    <!-- Juga</tr>
    <tfoot>
        <tr>
            <td>Juga</td>
            <td>$15</td>
        </tr>
    </tfoot>
</table>
```

## 3️⃣ colspan va rowspan

Katagni bir nechta ustun yoki qatorga cho'zish uchun foydalaniladi.

### colspan - Bir Nechta Ustun

```html
<table border="1">
    <tr>
        <!-- Bu katag 3 ta ustunni egallaydi -->
        <td colspan="3">Sarlavha</td>
    </tr>
    <tr>
        <td>Ustun 1</td>
        <td>Ustun 2</td>
        <td>Ustun 3</td>
    </tr>
</table>
```

### rowspan - Bir Nechta Qator

```html
<table border="1">
    <tr>
        <!-- Bu katag 2 ta qatorni egallaydi -->
        <td rowspan="2">Ustun 1</td>
        <td>Ustun 2</td>
    </tr>
    <tr>
        <td>Ustun 2 (satr 2)</td>
    </tr>
</table>
```

### Aralash Misol

<CodeBlock language="html" title="Colspan va Rowspan Misoli">
```html
<table border="1">
    <tr>
        <th colspan="2">Jami</th>
        <th>Narxi</th>
    </tr>
    <tr>
        <td rowspan="2">Kitoblar</td>
        <td>Qo'llanma 1</td>
        <td>$20</td>
    </tr>
    <tr>
        <td>Qo'llanma 2</td>
        <td>$25</td>
    </tr>
    <tr>
        <td colspan="2">Juga</td>
        <td>$45</td>
    </tr>
</table>
```
</CodeBlock>

## ✅ Best Practices

### 1. Semantik Taglarni Ishlatish

```html
<!-- ❌ Yomon - div bilan -->
<div>
    <div>Sarlavha 1</div>
    <div>Sarlavha 2</div>
</div>

<!-- ✅ Yaxshi - jadvalni taglar -->
<table>
    <thead>
        <tr>
            <th>Sarlavha 1</th>
            <th>Sarlavha 2</th>
        </tr>
    </thead>
</table>
```

### 2. <thead>, <tbody>, <tfoot> Ishlatish

```html
<!-- ❌ Yomon - shunga teglar yo'q -->
<table>
    <tr><th>Name</th></tr>
    <tr><td>Ali</td></tr>
</table>

<!-- ✅ Yaxshi - semantik qismlar -->
<table>
    <thead><tr><th>Name</th></tr></thead>
    <tbody><tr><td>Ali</td></tr></tbody>
</table>
```

### 3. Jadvalni Sarlavha Beya (Caption)

```html
<!-- ✅ Yaxshi - caption bor -->
<table>
    <caption>2026-yil Ma'lumotlari</caption>
    <thead>...</thead>
</table>
```

### 4. Accessibility uchun scope Atributi

```html
<!-- ✅ Screen reader'lar uchun yaxshi -->
<th scope="col">Ism</th>      <!-- Ustun sarlavhasi -->
<th scope="row">Javob</th>    <!-- Satr sarlavhasi -->
```

## ❌ Keng Uchraydigan Xatoliklar

### 1. Layout uchun Jadval

```html

    <tr>
        <td>Header</td>
    </tr>
    <tr>
        <td>Sidebar</td>
        <td>Main Content</td>
    </tr>
</table>

<!-- ✅ CSS uchun Flexbox yoki Grid -->
<header>Header</header>
<main>
    <aside>Sidebar</aside>
    <div>Main Content</div>
</main>
```

### 2. <th> Masalasi

```html
<!-- ❌ Yomon - hamma <td> -->
<tr>
    <td>Ism</td>
    <td>Narxi</td>
</tr>

<!-- ✅ Yaxshi - sarlavha <th> -->

    <th>Ism</th>
    <th>Narxi</th>

```

### 3. colspan/rowspan Noto'g'ri Ishlatish

```html

    <td colspan="5">Too many columns</td>
    <td>Column</td>  <!-- Extra! -->

<!-- ✅ Yaxshi - to'g'ri o'lcham -->
<tr>
    <td colspan="3">3 ta ustun</td>
    <td>1 ta ustun</td>
</tr>
```

## DOM Manipulation

HTML elementlarini o'zgartirishga DOM deyiladi:

```javascript
// Element topish
const element = document.getElementById("myId");
const elements = document.querySelectorAll(".myClass");

// Element o'zgartirish
element.textContent = "Yangi matn";
element.style.color = "red";
element.classList.add("active");

// Element qo'shish
const newElement = document.createElement("div");
newElement.textContent = "Yangi element";
document.body.appendChild(newElement);
```

## Events

Foydalanuvchi harakatlariga javob berish:

```javascript
const button = document.querySelector("button");

// Click event
button.addEventListener("click", () => {
  alert("Tugma bosildi!");
});

// Mouse over event
button.addEventListener("mouseover", () => {
  button.style.background = "blue";
});

// Form submit event
const form = document.querySelector("form");
form.addEventListener("submit", (e) => {
  e.preventDefault();
  console.log("Forma submitted!");
});
```

## Async/Await

API'dan ma'lumot olish:

```javascript
async function fetchData() {
  try {
    const response = await fetch("https://api.example.com/data");
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error("Xatolik:", error);
  }
}

fetchData();
```
