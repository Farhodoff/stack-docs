# BEM Metodologiyasi - CSS Nomlanish Standardi

BEM (Block Element Modifier) - CSS kod va HTML strukturi stiI va tartibini belgilaydigan metodologiya. Bu CSS'ni tashkilashtirilgan, o'qilgan va saqlanadi qiladi.

---

## 1️⃣ BEM Nima?

BEM uchta asosiy kontsepyaya asoslangan:

```
.block { }           /* Bloklarning bosh qismi */
.block__element { }  /* Blokning ichidagi element */
.block--modifier { } /* Blokning versiyasi yoki holati */
```

---

## 2️⃣ Block (Blok)

Blok - o'zi ichida to'la va qayta ishlatilishi mumkin olan bo'lagi:

```html
<!-- Button block'i -->
<button class="button">Bosing</button>

<!-- Karta block'i -->
<div class="card">
  <h2>Sarlavha</h2>
  <p>Matn</p>
</div>

<!-- Navbar block'i -->
<nav class="navbar">
  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>
</nav>
```

```css
.button {
  padding: 10px 20px;
  background-color: #2563eb;
  color: white;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

.card {
  background-color: white;
  border-radius: 8px;
  padding: 20px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

.navbar {
  background-color: #333;
  padding: 15px 0;
}
```

---

## 3️⃣ Element (Element)

Element - blok ichidagi qism. Ikki pastgi chiziq __ bilan ajratiladi:

```html
<div class="card">
  <h2 class="card__title">Sarlavha</h2>
  <p class="card__description">Tasnifi</p>
  <a href="#" class="card__link">Ko'proq o'qish</a>
</div>
```

```css
.card {
  background-color: white;
  border-radius: 8px;
  padding: 20px;
}

.card__title {
  font-size: 24px;
  color: #2563eb;
  margin-bottom: 10px;
}

.card__description {
  color: #666;
  line-height: 1.6;
  margin-bottom: 15px;
}

.card__link {
  color: #2563eb;
  text-decoration: none;
}

.card__link:hover {
  text-decoration: underline;
}
```

## 4️⃣ Modifier (O'zgartirgich)

Modifier - blok yoki elementning holati yoki versiyasi. Bitta chiziq - bilan ajratiladi:

```html
<!-- Turli tugma turlari -->
<button class="button">Normal</button>
<button class="button button--primary">Asosiy</button>
<button class="button button--secondary">Ikkinchi</button>
<button class="button button--danger">Xavf</button>
<button class="button button--disabled">O'chirilgan</button>

<!-- Karta holatlari -->
<div class="card card--featured">
  <h2 class="card__title">Sarlavha</h2>
  <p class="card__description">Tasnifi</p>
</div>

<div class="card card--small">
  <h2 class="card__title card__title--small">Kichik Sarlavha</h2>
  <p class="card__description">Tasnifi</p>
</div>
```

```css
/* Blok modifierlari */
.button {
  padding: 10px 20px;
  background-color: #e5e7eb;
  color: #333;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  transition: all 0.3s ease;
}

.button--primary {
  background-color: #2563eb;
  color: white;
}

.button--primary:hover {
  background-color: #1d4ed8;
}

.button--secondary {
  background-color: #059669;
  color: white;
}

.button--danger {
  background-color: #dc2626;
  color: white;
}

.button--disabled {
  background-color: #d1d5db;
  color: #9ca3af;
  cursor: not-allowed;
  opacity: 0.6;
}

/* Element modifierlari */
.card {
  background-color: white;
  border-radius: 8px;
  padding: 20px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

.card--featured {
  border: 2px solid #2563eb;
  box-shadow: 0 4px 16px rgba(37, 99, 235, 0.2);
  padding: 25px;
}

.card--small {
  padding: 15px;
  box-shadow: 0 1px 4px rgba(0, 0, 0, 0.1);
}

.card__title {
  font-size: 24px;
  color: #2563eb;
  margin-bottom: 10px;
}

.card__title--small {
  font-size: 18px;
}
```

## 5️⃣ BEM Nesting - Bosh va Ichki Bloklar

Bitta blok boshqa blok ichida bo'lishi mumkin:

```html
<div class="navbar">
  <div class="navbar__logo">Logo</div>

  <div class="navbar__menu">
    <a href="#" class="menu__item">Item 1</a>
    <a href="#" class="menu__item">Item 2</a>
  </div>

  <div class="navbar__user-menu">
    <button class="button button--small">Profil</button>
  </div>
</div>
```

```css
.navbar {
  background-color: #333;
  padding: 15px 20px;
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.navbar__logo {
  font-size: 24px;
  font-weight: bold;
  color: white;
}

.navbar__menu {
  display: flex;
  gap: 30px;
}

.navbar__user-menu {
  display: flex;
  gap: 10px;
}

/* Menu block - BEM bilan */
.menu__item {
  color: white;
  text-decoration: none;
  padding: 8px 0;
  border-bottom: 2px solid transparent;
  transition: all 0.3s ease;
}

.menu__item:hover {
  border-bottom-color: #2563eb;
}
```

## 7️⃣ BEM Nomlanish Qoidalari

Doim qo'llash kerak bo'lgan qoidalar:

```css
/* ✅ YAXSHI - Aniq va tashkilashtirilgan */
.navbar { }
.navbar__logo { }
.navbar__menu { }
.navbar__menu-item { }
.button { }
.button--primary { }

/* ❌ YOMON - Buzilgan BEM */
.navbar_logo { }           /* Bitta chiziq */
.navbar---menu { }         /* Uch chiziq */
.nav_bar__logo { }         /* Noto'g'ri blok nomi */
.button-primary { }        /* Bitta chiziq */

/* ❌ YOMON - Katta CSS nesting */
.card .card-title { }      /* BEM emas, CSS nesting */
.menu > li > a { }         /* Katta nesting */

/* ✅ YAXSHI - BEM */
.card { }
.card__title { }
.menu { }
.menu__item { }
.menu__link { }
```

## BEM Cheat Sheet

| Konsepatyi | Namuna | Maqsadi |
|-----------|--------|---------|
| Block | `.card` | Asosiy komponent |
| Element | `.card__title` | Blokning qismi |
| Modifier | `.card--featured` | Holat yoki versiya |
| Multiple | `.button--primary--large` | Bir nechta modifikator |

---

## 🎯 BEM Qisqacha

- **Block** → `.blockname` - mustaqil komponent
- **Element** → `.blockname__elementname` - bloking qismi
- **Modifier** → `.blockname--modifiername` - holat yoki versiya
- **TAVSIYA:** BEM katta va murakkab loyihalar uchun eng yaxshi!
