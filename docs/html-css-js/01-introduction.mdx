# Vebning asosiy ustunlari

Internetdagi har bir veb-sayt, qanchalik murakkab bo'lmasin, uchta asosiy texnologiya yordamida quriladi: **HTML**, **CSS** va **JavaScript**. Ularni veb-sahifaning **suyagi**, **terisi** va **mushaklari** deb tasavvur qilishingiz mumkin.

Ushbu o'quv yo'nalishida siz zamonaviy, moslashuvchan va interaktiv foydalanuvchi interfeyslarini yaratish uchun ushbu uchta asosiy tilni o'zlashtirasiz.

-----

## 1\. HTML: Tuzilishi (Suyak)

**HTML (HyperText Markup Language)** — veb-brauzerda ko'rsatish uchun mo'ljallangan hujjatlar uchun standart belgilash tilidir. U saytning asosiy tuzilishini ta'minlaydi.

  - **Maqsadi:** Tarkib va joylashuvni belgilaydi (sarlavhalar, paragraflar, rasmlar, havolalar).
  - **O'xshatish:** Uyning g'ishtlari va poydevori.

\<CodeBlock language="html" title="index.html"\>

```html
&lt;!DOCTYPE html&gt;
&lt;html lang=&quot;uz&quot;&gt;
&lt;head&gt;
    &lt;title&gt;Mening birinchi veb-saytim&lt;/title&gt;
&lt;/head&gt;
&lt;body&gt;
    &lt;h1&gt;Fullstack Docs-ga xush kelibsiz!&lt;/h1&gt;
    &lt;p&gt;Bu veb-saytni tavsiflovchi matnli paragraf.&lt;/p&gt;
    &lt;button&gt;Meni bosing!&lt;/button&gt;
&lt;/body&gt;
&lt;/html&gt;
```

\</CodeBlock\>

-----

## 2\. CSS: Taqdimot (Teri)

**CSS (Cascading Style Sheets)** veb-sahifalarni shakllantirish va tartibga solish uchun ishlatiladi. HTML brauzerga kontent *nima* ekanligini aytsa, CSS brauzerga u *qanday* ko'rinishi kerakligini aytadi.

  - **Maqsadi:** Ranglar, shriftlar, masofalar, animatsiyalar va moslashuvchan dizaynni boshqaradi.
  - **O'xshatish:** Uyning bo'yog'i, interyer dizayni va arxitekturasi.

\<CodeBlock language="css" title="styles.css"\>

```css
body {
    font-family: &#39;Inter&#39;, sans-serif;
    background-color: #f4f4f9;
    color: #333;
}

h1 {
color: \#2563eb; /\* Chiroyli ko'k rang \*/
text-align: center;
}

button {
background-color: \#2563eb;
color: white;
padding: 10px 20px;
border-radius: 5px;
border: none;
cursor: pointer;
}

button:hover {
background-color: \#1d4ed8;
}

```

</CodeBlock>

## 3\. JavaScript: Xulq-atvor (Mushaklar)

**JavaScript (JS)** — bu dinamik yangilanadigan tarkib yaratish, multimediani boshqarish, tasvirlarni harakatga keltirish va boshqa ko'p narsalarni amalga oshirish imkonini beruvchi skript tilidir. U sizning statik HTML va CSS-ga "jon" kiritadi.

  - **Maqsadi:** Interaktivlik, mantiq, ma'lumotlarni qabul qilish va dinamik UI yangilanishlarini qo'shadi.
  - **O'xshatish:** Uyning elektr energiyasi, suv tizimi va aqlli uy tizimi.

## Qanday qilib ular birgalikda ishlaydi

Foydalanuvchi veb-saytga tashrif buyurganida:

1.  Brauzer avval **HTML** faylini so'raydi va uni yuqoridan pastga o'qishni boshlaydi.
2.  Brauzer HTML ichida `<link>` tegini ko'rsa, **CSS** faylini yuklaydi va uslublarni qo'llaydi.
3.  Brauzer `<script>` tegiga duch kelganida, u **JavaScript** kodini bajaradi, bu esa o'z navbatida HTML yoki CSS-ni jarayon davomida o'zgartirishi mumkin.
