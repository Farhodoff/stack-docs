# Mobile-first Yondashuv va Responsive Dizayn

Mobile-first - mobil ekranga mos qilib, keyin katta ekranlarga o'zgartirasiz. Bu strategiya faqat zarur CSS'ni yuklaydi va barcha qurilmada yaxshi ishlaydi.

---

## 1️⃣ Mobile-first nima?

Ikkita yondashuv:

```css
/* ❌ Desktop-first (eski usul) */
.container {
  width: 1200px;  /* Desktop uchun */
}

@media (max-width: 768px) {
  .container {
    width: 100%;  /* Mobil uchun */
  }
}

/* ✅ Mobile-first (yangi usul) */
.container {
  width: 100%;  /* Mobil uchun asosiy */
}

@media (min-width: 768px) {
  .container {
    width: 1200px;  /* Planshet uchun */
  }
}
```

---

## 2️⃣ Viewport Meta Tag

HTML'nin `<head>`'ga qo'shilishi shart:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

Bu qoida quyidagini ta'minlaydi:
- Qurilma kengligiga mos qilish
- Boshlang'ich masshtab 1 (100%)

## 3️⃣ Breakpoint'lar

Turli ekran o'lchamlariga mos CSS:

```css
/* Mobil (default) - 0px va yuqori */
.box {
  font-size: 14px;
  column-count: 1;
}

/* Kichik planshet (480px va yuqori) */
@media (min-width: 480px) {
  .box {
    font-size: 16px;
  }
}

/* Planshet (768px va yuqori) */
@media (min-width: 768px) {
  .box {
    font-size: 18px;
    column-count: 2;
  }
}

/* Katta planshet (1024px va yuqori) */
@media (min-width: 1024px) {
  .box {
    font-size: 20px;
    column-count: 3;
  }
}

/* Desktop (1200px va yuqori) */
@media (min-width: 1200px) {
  .box {
    max-width: 1200px;
    margin: 0 auto;
  }
}

/* Katta desktop (1440px va yuqori) */
@media (min-width: 1440px) {
  .box {
    max-width: 1400px;
  }
}
```

## 6️⃣ Responsive Images

Rasmlarni responsiv qilish:

```css
/* Asosiy - img barcha ekranda 100% */
img {
  max-width: 100%;
  height: auto;
  display: block;
}

/* Ekran katta bo'lganda aniq o'lchami */
@media (min-width: 1024px) {
  img {
    max-width: 600px;
    width: 100%;
  }
}
```

HTML'da srcset ishlatish:

```html
<!-- Turli o'lchamlarda turli rasmlar -->
<img
  src="image-small.jpg"
  srcset="
    image-small.jpg 480w,
    image-medium.jpg 768w,
    image-large.jpg 1200w
  "
  sizes="(max-width: 480px) 100vw,
         (max-width: 768px) 80vw,
         (max-width: 1200px) 60vw,
         1200px"
  alt="Rasm tasnifi"
>
```

Picture element:

```html
<picture>
  <!-- Mobil uchun */
  <source media="(max-width: 600px)" srcset="mobile.jpg">
  <!-- Planshet uchun -->
  <source media="(max-width: 1024px)" srcset="tablet.jpg">
  <!-- Desktop uchun -->
  <img src="desktop.jpg" alt="Rasm">
</picture>
```

## 7️⃣ Flexible Grid System

Mobile-first grid:

```css
/* MOBIL - 1 ustun */
.grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

/* TABLET - 2 ustun */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* DESKTOP - 3 ustun */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* LARGE - 4 ustun */
@media (min-width: 1440px) {
  .grid {
    grid-template-columns: repeat(4, 1fr);
  }
}
```

## 1️⃣0️⃣ Media Query Patterns

```css
/* Orientation - Telefonning yo'nalishi */
@media (orientation: portrait) {
  /* Vertikal telefonlar */
}

@media (orientation: landscape) {
  /* Gorizontal telefonlar */
}

/* Touch Device'lar */
@media (hover: none) {
  /* Touch qurilmalar - hover yo'q */
  button {
    padding: 20px;  /* Katta tugma */
  }
}

/* Pixel Density - Retina */
@media (-webkit-min-device-pixel-ratio: 2) {
  /* Retina ekranlari */
}

/* Dark Mode */
@media (prefers-color-scheme: dark) {
  body {
    background-color: #1a1a1a;
    color: #ffffff;
  }
}

/* Reduced Motion */
@media (prefers-reduced-motion: reduce) {
  * {
    animation: none !important;
    transition: none !important;
  }
}
```

@media (min-width: 768px) {

  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .md\:grid-cols-2 {

  }
}

## Mobile-first Cheat Sheet

| Breakpoint | Min-width | Ekran |
|-----------|-----------|-------|
| `xs` | 0px | Mobil |
| `sm` | 480px | Kichik mobil |
| `md` | 768px | Planshet |
| `lg` | 1024px | Katta planshet |
| `xl` | 1200px | Desktop |
| `2xl` | 1440px | Katta desktop |

---

## 🎯 Mobile-first Qisqacha

- **Mobile-first** → Mobil uchun asosiy CSS, keyin katta ekranlarga o'zgartiriladi
- `min-width` → Breakpoint'lar uchun
- `max-width: 100%` → Rasmlar va kontentni responsive qilish
- **TAVSIYA:** Har doim mobil uchun boshlang'ich, keyin katta ekranlarga o'zgartirilsin!
