# CSS Layout - Grid

**CSS Grid** - qatorlar va ustunlardan iborat sof 2D joylashtirishning samatli usuli. Flexbox'dan farqli o'laroq, Grid bir vaqtning o'zida qatorlar VA ustunlarni boshqaradi.

---

## Grid vs Flexbox

| Xususiyat | Flexbox | Grid |
|-----------|---------|------|
| O'lcham | 1D (qator yoki ustun) | 2D (qator + ustun) |
| Joylashturuvi | Flexible | qat'iy |
| Ishlatish | Komponentlar | Saytning bosh shabloni |

---

## 1️⃣ Display: Grid

Grid'ni aktivlashtirish:

```css
.container {
  display: grid;
}
```

---

## 2️⃣ Grid Template Columns (Ustunlar)

Nechta va qanday o'lchamda ustun bo'lish:

```css
.container {
  display: grid;

  /* 3 ta teng ustun */
  grid-template-columns: 1fr 1fr 1fr;
}

/* Alternativ */
.container {
  grid-template-columns: repeat(3, 1fr);
}

/* Turli o'lchamlar */
.container {
  grid-template-columns: 1fr 2fr 1fr;
}

/* Piksel bilan */
.container {
  grid-template-columns: 200px 1fr 150px;
}

/* Auto fit */
.container {
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}
```

**fr (fraction unit):** Bo'sh joyni teng taqsimlash

---

## 3️⃣ Grid Template Rows (Qatorlar)

Qatorlarni boshqarish:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;

  /* 3 ta qator */
  grid-template-rows: 100px 200px 150px;
}

/* Teng qatorlar */
.container {
  grid-template-rows: repeat(3, 1fr);
}

/* Auto balandlik */
.container {
  grid-template-rows: auto 1fr auto;
}
```

---

## 4️⃣ Gap (Bo'sh Joy)

Qatorlar va ustunlar orasida bo'sh joy:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;

  /* Hamma bo'sh joy */
  gap: 20px;
}

/* Alohida */
.container {
  row-gap: 20px;      /* Qatorlar orasida */
  column-gap: 15px;   /* Ustunlar orasida */
}
```

---

## 5️⃣ Grid Column (Ustun Span)

Element nechta ustunda o'tishin:

```css
.item {
  /* By default - bitta */
  grid-column: 1;
}

.item.wide {
  /* 2 ta ustun */
  grid-column: span 2;
}

.item.full {
  /* Barcha ustunlar */
  grid-column: 1 / -1;
}
```

---

## 6️⃣ Grid Row (Qator Span)

Element nechta qatorda o'tishin:

```css
.item {
  grid-row: span 2;    /* 2 ta qator */
}

.item.tall {
  grid-row: 1 / 4;     /* 1-qatordan 4-gacha */
}
```

---

## 7️⃣ Justify Items va Align Items

Har bir cell'da elementni joylashtirish:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;

  /* Horizontal */
  justify-items: center;
}

.container {
  /* Vertical */
  align-items: center;
}

.container {
  /* Ikkalasi */
  place-items: center;
}
```

---

## 8️⃣ Justify Content va Align Content

Barcha grid'ni container'da joylashtirish:

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  height: 500px;

  /* Horizontal */
  justify-content: center;
}

.container {
  /* Vertical */
  align-content: center;
}
```

## Grid Cheat Sheet

| Xususiyat | Misol |
|-----------|-------|
| `display: grid` | Grid'ni aktivlashtirish |
| `grid-template-columns` | `repeat(3, 1fr)` |
| `grid-template-rows` | `100px 1fr auto` |
| `gap` | `20px` |
| `grid-column` | `span 2` |
| `grid-row` | `span 3` |
| `justify-items` | `center` |
| `align-items` | `center` |

---

## 🎯 Grid Qisqacha

- **Grid:** 2D joylashuv (qator + ustun)
- **`grid-template-columns`:** Ustunlarni belgilash
- **`grid-template-rows`:** Qatorlarni belgilash
- **`gap`:** Bo'sh joy
- **`grid-column: span 2`:** 2 ustunni egallash
- **`repeat(3, 1fr)`:** 3 ta teng ustun
- **TAVSIYA:** Saytning bosh shabloni uchun Grid, komponentlar uchun Flexbox
