# Havolalar va Rasmlar

Havolalar (`<a>`) va rasmlar (`<img>`) veb-sahifalarni **interaktiv va vizual** qiladi. Bu darsda siz havolalarni yaratishni, rasmlarni qo'shishni va best practice'larni o'rganasiz.

---

## 1️⃣ Havolalar (<a> Tegi)

`<a>` tegi (anchor) **boshqa sahifalarga, fayllariga yoki veb-saytlarga** havola yaratadi.

### Asosiy A Tegi

<CodeBlock language="html" title="Oddiy Havola">
```html
<!-- Boshqa saytga havola -->
<a href="https://www.example.com">Example.com'ni tashrif buyuring</a>

<!-- Shu sayti ichidagi sahifaga havola -->
<a href="/about">Haqida sahifasi</a>

<!-- O'zbekiston Respublikasining rasmiy saytiga -->
<a href="https://gov.uz">O'zbekiston Hukumati</a>
```
</CodeBlock>

### A Tegi Atributlari

#### `href` - Havola Manzili

```html
<!-- Dastlabki URL (absolute URL) -->
<a href="https://google.com">Google</a>

<!-- Nisbiy URL (relative URL) -->
<a href="/pages/about.html">Haqida</a>

<!-- Bir fayl yoki papka -->
<a href="../index.html">Bosh sahifaga qaytish</a>

<!-- Email manzili -->
<a href="mailto:info@example.com">Biz bilan bog'lanish</a>

<!-- Telefon raqami -->
<a href="tel:+998901234567">+998 90 123 45 67</a>
```

#### `target` - Havola Qayerda Ochiladi

```html
<!-- Hozirgi tabtada ochish (default) -->
<a href="https://google.com">Google</a>

<!-- Yangi tabtada ochish -->
<a href="https://google.com" target="_blank">Google (Yangi tab)</a>

<!-- Yangi oynada ochish -->
<a href="https://google.com" target="_blank">Google (Yangi oyna)</a>

<!-- Parent frameda ochish (nadirdan foydalaniladi) -->
<a href="https://google.com" target="_parent">Parent</a>

<!-- To'liq brauzerda ochish (nadirdan foydalaniladi) -->
<a href="https://google.com" target="_top">Top</a>
```

#### `title` - Havola Tavsifi

Foydalanuvchi havolaning ustiga o'tkaonda tooltip ko'rsatiladi:

```html
<a href="https://mozilla.org" title="Mozilla Foundation - Open Source">
    Mozilla
</a>

<!-- Mobilda title ko'rsatilmaydi -->
<a href="/about" title="Kompaniya haqida batafsil ma'lumot">
    Haqida
</a>
```

#### `rel` - Havola Munosabati

```html
<!-- Tashqi saytga nofollow -->
<a href="https://example.com" rel="nofollow">
    Boshqa Sayt
</a>

<!-- Tashqi saytga rel="external" -->
<a href="https://example.com" rel="external" target="_blank">
    Tashqi sayt
</a>

<!-- Rasm faylini olish -->
<a href="/files/document.pdf" rel="download">
    PDF'ni Yuklab Olish
</a>

<!-- Bir nechta rel atributlari -->
<a href="https://example.com" rel="nofollow external">
    Link
</a>
```

#### `download` - Fayl Yuklab Olish

```html
<!-- Fayl yuklab olish (boshqotirmadan) -->
<a href="/files/document.pdf" download>
    PDF'ni Yuklab Olish
</a>

<!-- Boshqotirma bilan yuklab olish -->
<a href="/files/document.pdf" download="Mening_Dokumentim">
    PDF'ni Yuklab Olish
</a>

<!-- Rasm yuklab olish -->
<a href="/images/photo.jpg" download="Mening_Fotom">
    Fotoni Yuklab Olish
</a>
```

## 2️⃣ Ichki Navigatsiya (Anchor Links)

Bir sahifa ichidagi har xil bo'lim o'rtasida havolalar yaratishingiz mumkin.

<CodeBlock language="html" title="Anchor Links Misoli">
```html
<!DOCTYPE html>
<html lang="uz">
<head>
    <title>Anchor Links</title>
</head>
<body>
    <!-- Navigatsiya Menyu -->
    <nav>
        <ul>
            <li><a href="#section1">Bo'lim 1</a></li>
            <li><a href="#section2">Bo'lim 2</a></li>
            <li><a href="#section3">Bo'lim 3</a></li>
        </ul>
    </nav>

    <!-- Bo'lim 1 -->
    <section id="section1">
        <h2>Bo'lim 1</h2>
        <p>Bu birinchi bo'lim...</p>
    </section>

    <!-- Bo'lim 2 -->
    <section id="section2">
        <h2>Bo'lim 2</h2>
        <p>Bu ikkinchi bo'lim...</p>
    </section>

    <!-- Bo'lim 3 -->
    <section id="section3">
        <h2>Bo'lim 3</h2>
        <p>Bu uchinchi bo'lim...</p>
        <!-- Yuqoriga qaytish -->
        <a href="#section1">Bosh sahifaga qaytish</a>
    </section>
</body>
</html>
```
</CodeBlock>

**Qanday Ishlaydi:**
1. `id="section1"` - Bo'limni identifikator beradi
2. `<a href="#section1">` - O'sha bo'limga havola yaratadi
3. Havola bosilganda, brauzer o'sha bo'limga ko'chadi

## 3️⃣ Rasmlar (<img> Tegi)

`<img>` tegi sahifaga rasmlar qo'shadi.

### Asosiy IMG Tegi

<CodeBlock language="html" title="Oddiy Rasm">
```html
<!-- Eng sodda rasm -->
<img src="/images/rasm.jpg">

<!-- Rasm o'lchamlari bilan -->
<img src="/images/rasm.jpg" width="300" height="200">

<!-- Rasm tavsifi (alt text) -->
<img src="/images/rasm.jpg" alt="Toshkent shahri yosh yoshlari">
```
</CodeBlock>

### IMG Tegi Atributlari

#### `src` - Rasm Faylining Manzili

```html
<!-- Dastlabki URL -->
<img src="https://example.com/rasm.jpg">

<!-- Nisbiy URL -->
<img src="/images/rasm.jpg">
<img src="../images/rasm.jpg">

<!-- Data URL (Base64 kodlangan rasm) -->
<img src="data:image/png;base64,iVBORw0KGg...">
```

#### `alt` - Rasm Tavsifi (Accessibility)

**Juda muhim!** Alt matn:
- Screen reader'larga o'qiladi
- Rasm yuklana olmasa ko'rsatiladi
- SEO uchun muhim

```html
<!-- ❌ Yomon - alt yo'q -->
<img src="/rasm.jpg">

<!-- ✅ Yaxshi - batafsil alt tavsifi -->
<img src="/rasm.jpg" alt="Toshkent shahri Amir Temur xiyoboni">

<!-- Dekorativ rasm - bo'sh alt -->
<img src="/logo-pattern.png" alt="">
```

#### `title` - Rasm Ustiga Tooltip

```html
<img src="/rasm.jpg"
     alt="Toshkent"
     title="Bu Toshkent shahri, O'zbekistonning poytaxti">
```

#### `width` va `height` - O'lcham

```html
<!-- Pikselda o'lcham -->
<img src="/rasm.jpg" width="300" height="200">

<!-- Faqat width (height avtomatik) -->
<img src="/rasm.jpg" width="300">

<!-- Responsivlik uchun percentda -->
<img src="/rasm.jpg" style="width: 100%; height: auto;">
```

#### `loading` - Rasm Yuklanish Turi

```html
<!-- Lazy loading (sahifa scroll qilinganda yuklanadi) -->
<img src="/rasm.jpg" loading="lazy">

<!-- Shuni bilan yuklanadi (default) -->
<img src="/rasm.jpg" loading="eager">
```

## 4️⃣ Responsive Rasmlar

Turli ekran o'lchamlari uchun turli rasmlarni ko'rsatishingiz mumkin.

### Picture Element

<CodeBlock language="html" title="Responsive Rasmlar">
```html
<!-- Ekran o'lchamiga qarab turli rasmlar -->
<picture>
    <!-- Telefon uchun -->
    <source srcset="/images/rasm-small.jpg" media="(max-width: 600px)">

    <!-- Planshet uchun -->
    <source srcset="/images/rasm-medium.jpg" media="(max-width: 1200px)">

    <!-- Desktop uchun -->
    <img src="/images/rasm-large.jpg" alt="Responsive rasm misoli">
</picture>

<!-- Turli format'larni qo'llab-quvvatlash -->
<picture>
    <!-- Zamonaviy WebP format -->
    <source srcset="/images/rasm.webp" type="image/webp">

    <!-- Eski format -->
    <img src="/images/rasm.jpg" alt="Rasm">
</picture>
```
</CodeBlock>

### Srcset Atributi

```html
<!-- Pixeldensity uchun turli rasmlar -->
<img src="/rasm.jpg"
     srcset="/rasm-1x.jpg 1x, /rasm-2x.jpg 2x"
     alt="Retina displey uchun optimizatsiya">

<!-- Ekran o'lchamiga qarab turli rasmlar -->
<img src="/rasm.jpg"
     srcset="/rasm-small.jpg 600w,
             /rasm-medium.jpg 1000w,
             /rasm-large.jpg 1600w"
     sizes="(max-width: 600px) 100vw,
            (max-width: 1000px) 50vw,
            33vw"
     alt="Responsive rasm">
```

```html

    <!-- Havolalar Bo'limi -->
    <h2>Turli Xil Havolalar</h2>

        <a href="https://www.gov.uz" target="_blank">
            O'zbekiston Hukumati Saytiga O'tish
        </a>
    </p>

    <!-- Shu saytning ichidagi sahifaga -->
    <p>
        <a href="/about.html">Haqida sahifasi</a>
    </p>

    <!-- Email havola -->
    <p>
        <a href="mailto:info@example.com">
            Biz bilan bog'lanish: info@example.com
        </a>
    </p>

    <!-- Telefon havola -->
    <p>
        <a href="tel:+998901234567">+998 90 123 45 67</a>
    </p>

    <hr>

    <!-- Rasmlar Bo'limi -->
    <h2>Rasmlar</h2>

    <!-- Oddiy rasm -->
    <p>
        <img src="/images/sample.jpg"
             alt="Toshkent shahri"
             width="300"
             height="200">
    </p>

    <!-- Responsive rasm -->
    <picture>
        <source srcset="/images/small.jpg" media="(max-width: 600px)">
        <source srcset="/images/large.jpg" media="(min-width: 601px)">
        <img src="/images/default.jpg" alt="Responsive rasm">
    </picture>

    <hr>

    <!-- Anchor Links -->
    <h2>Sahifa Ichidagi Navigatsiya</h2>
    <ul>
        <li><a href="#intro">Kirish</a></li>
        <li><a href="#content">Asosiy Kontent</a></li>
        <li><a href="#conclusion">Xulosa</a></li>
    </ul>

    <!-- Bo'lim 1 -->
    <section id="intro">
        <h3>Kirish</h3>
        <p>HTML'da havolalar va rasmlar juda muhim element...</p>
    </section>

    <!-- Bo'lim 2 -->
    <section id="content">
        <h3>Asosiy Kontent</h3>
        <p>Rasmlar sahifalarni yanada chiroyli qiladi...</p>
    </section>

    <!-- Bo'lim 3 -->
    <section id="conclusion">
        <h3>Xulosa</h3>
        <p>Havolalar va rasmlar veb-saytning asosiy qismidir...</p>
        <a href="#intro">Yuqoriga qaytish</a>
    </section>
</body>
</html>
```
</CodeBlock>

## ✅ Best Practices

### 1. Alt Atributini Har Doim Ishlatish

```html
<!-- ❌ Yomon - alt yo'q -->
<img src="/rasm.jpg">

<!-- ✅ Yaxshi - batafsil alt -->
<img src="/rasm.jpg" alt="Yoshlar birgalikda dasturlash qilyapti">

<!-- Dekorativ rasm -->
<img src="/gradient.png" alt="">
```

### 2. Responsive Rasmlar

```html
<!-- ❌ Yomon - fixed o'lcham -->
<img src="/rasm.jpg" width="800" height="600">

<!-- ✅ Yaxshi - responsive -->
<img src="/rasm.jpg"
     alt="Responsive rasm"
     style="max-width: 100%; height: auto;">
```

### 3. Havola Matnini Mantiqiy Qilish

```html
<!-- ❌ Yomon -->
<a href="/about">Bu yerga bosing</a>

<!-- ✅ Yaxshi -->
<a href="/about">Haqida sahifasi</a>
```

### 4. Target Atributini Ehtiyot Bilan Ishlatish

```html
<!-- ❌ Juda ko'p yangi tab ochish -->
<a href="..." target="_blank">Link 1</a>
<a href="..." target="_blank">Link 2</a>

<!-- ✅ Tashqi saytlar uchun faqat -->
<a href="https://example.com" target="_blank">Tashqi sayt</a>
```

### 5. Rasm Fayl O'lchamini Optimize Qilish

```html
<!-- ❌ Juda katta fayl (5 MB) -->
<img src="/rasm-original.jpg">

<!-- ✅ Optimallashtirgan (300 KB) -->
<img src="/rasm-optimized.jpg" alt="...">
```

## ❌ Keng Uchraydigan Xatoliklar

### 1. Alt Tavsifi Yo'q

```html
<!-- ❌ Yomon -->
<img src="/rasm.jpg">

<!-- ✅ Yaxshi -->
<img src="/rasm.jpg" alt="Rasm tavsifi">
```

### 2. Noto'g'ri Href

```html
<!-- ❌ Yomon -->
<a href="about.html">Haqida</a>  <!-- Fayl topilmadi -->

<!-- ✅ Yaxshi -->
<a href="/pages/about.html">Haqida</a>
```

### 3. Anchor Link Xatosi

```html
<!-- ❌ Yomon -->
<a href="#section-1">Go to Section</a>
<section id="Section-1">  <!-- ID not matching -->

<!-- ✅ Yaxshi -->
<a href="#section-1">Go to Section</a>
<section id="section-1">  <!-- Exact match -->
```

### 4. Buyuk Rasmlar Yuklanish Vaqtiga Sabab Bolish

```html
<!-- ❌ Yomon - 10 MB rasm -->
<img src="/huge-image.jpg">

<!-- ✅ Yaxshi - Optimallashtirgan -->
<img src="/optimized-image.jpg" loading="lazy">
```
